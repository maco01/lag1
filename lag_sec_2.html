第II節
4次方程式の解法について．

26．カルダノの同時代人であり，かつその弟子でもあったルイ・フェラーリは，4次方程式の一般的な解法を最初に見出した人物として知られている．彼の解法は，与えられた方程式の項を2つの辺に振り分け，しかる後に両辺へ同一の量を加えることで，双方の辺から平方根を抽出できるようにするものである．これによって方程式は2次へと簡約される． この手法は，本目的のために考案されたあらゆる方法の中でも，最も巧妙なものと見なされ，デカルト以前の解析学者たちによって広く採用されてきた．しかし，この著名な幾何学者は，フェラーリの手法に代えて，決して単純でも直接的でもないものの，ある側面において方程式の性質により適合している別の方法を採用すべきだと考えた． この方法こそが，今日ではほとんどの著者たちが採用しているものである． したがって，我々はまず，この2つの方法を順に検討し，その後，この種の方程式の解法として知られている他の方法に進むことにしよう．その中でも特に，チルンハウス氏およびオイラー氏とベズー氏の方法は注目に値する．

まずフェラーリに従って，与えられた方程式が第2項を欠いていると仮定する（よく知られているように常にそのような形にできる）．その場合，方程式は次のように表される： \[ x^{4} + n x^{2} + p x + q = 0. \] このとき，第1項を除くすべての項を右辺に移し，さらに両辺に同じ量 \(2y x^{2} + y^{2}\) を加えると（ここで \(y\) は未定数），次の式が得られる： \[ x^{4} + 2y x^{2} + y^{2} = (2y - n)x^{2} - p x + y^{2} - q. \] この式の左辺は明らかに \(x^{2} + y\) の平方であるから，あとは右辺も平方にすることが課題となる．よく知られているように，それには，\(-p x\) の係数の半分の平方が，他の2項の係数の積に一致しなければならない．すなわち，次の条件が得られる： \[ \frac{p^{2}}{4} = (2y - n)(y^{2} - q). \] これより，次の3次方程式が得られる： \[ y^{3} - \frac{n}{2} y^{2} - q y + \frac{4nq - p^{2}}{8} = 0. \] この方程式を解いて \(y\) の値が得られたと仮定すれば，与えられた方程式の右辺は次のように書ける： \[ (2y - n)\left[x - \frac{p}{2(2y - n)}\right]^{2}. \] したがって，両辺の平方根を取ると， \[ x^{2} + y = \left[x - \frac{p}{2(2y - n)}\right] \sqrt{2y - n} \] となり，未知数 \(x\) は2次方程式に帰着されるので，もはや困難はない． 簡単のため， \[z = \sqrt{2y - n}\] とおけば，次の式が得られる： \[ x^{2} - z x + y + \frac{p}{2z} = 0. \] ここから直ちに次の式が導かれる： \[ x = \frac{z + \sqrt{z^{2} - \dfrac{2p}{z} - 4y}}{2}, \] あるいは \(z\) の値を代入して， \[ x = \frac{\sqrt{2y - n} + \sqrt{-2y - n - \dfrac{2p}{\sqrt{2y - n}}}}{2}. \] この式において，2つの平方根に正負の符号を順に与えることで，与えられた方程式の4つの根すべてが得られる．

27．まず注目すべきは，この方法は与えられた方程式の第2項の消去を必要とするものではなく，次のような完全形の方程式に対しても直接的に適用可能であるという点である： \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0. \] この場合，左辺が単に \(x^{2} + y\) の平方ではなく， \(x^{2} + \frac{m x}{2} + y\) の平方であると仮定する． 実際，先ほどと同様に，末尾の3項を右辺へ移項し，両辺に次の量を加える： \[ \left(2y + \frac{m^{2}}{4}\right)x^{2} + m y x + y^{2}. \] すると次の等式が得られる： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2} = \left(2y + \frac{m^{2}}{4} - n\right)x^{2} + (m y - p)x + y^{2} - q. \] 右辺も平方にするためには， \[ \left(\frac{m y - p}{2}\right)^{2} = \left(2y + \frac{m^{2}}{4} - n\right)(y^{2} - q). \] これより，次の3次方程式が得られる： \[ y^{3} - \frac{n}{2} y^{2} + \frac{m p - 4 q}{4} y + \frac{(4n - m^{2}) q - p^{2}}{8} = 0. \] \(y\)の値はこの方程式によって定める．これ以後，この方程式を簡約方程式と呼ぶことにする．簡単のため \[ z = \sqrt{2y + \frac{m^{2}}{4} - n} \] とおけば，次のように書ける： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2} = z^{2} \left(x + \frac{m y - p}{2 z^{2}}\right)^{2}. \] 両辺の平方根を取ると， \[ x^{2} + \frac{m x}{2} + y = z x + \frac{m y - p}{2 z}, \] すなわち， \[ x^{2} + \left(\frac{m}{2} - z\right)x + y - \frac{m y - p}{2 z} = 0. \] ここから最終的に次の式が得られる： \[ x = \frac{1}{2} \left[ z - \frac{m}{2} + \sqrt{ z^{2} - m z + \frac{m^{2}}{4} - 4 y + \frac{2(m y - p)}{z} } \right]. \] そして \(z\) の値を代入すれば， \[ x = \frac{1}{2} \left( - \frac{m}{2} + \sqrt{2y + \frac{m^{2}}{4} - n} + \sqrt{ -2y + \frac{m^{2}}{2} - n - \frac{\dfrac{1}{4} m^{3} - m n + 2 p}{\sqrt{2y + \dfrac{m^{2}}{4} - n}} } \right). \] この式において，2つの平方根に順に正負の符号を与えることで，やはり与えられた方程式の4つの根が得られる．

28．簡約方程式は \(y\) についての3次方程式であるから，必ず3つの根をもつ．それぞれの根は \(x\) を表す式に代入可能であり，さらにその式には2つの平方根が含まれているため，符号の取り方によって \(x\) の値は合計で12通り得られる．したがって，この解法は実質的に12次の方程式を解いているに等しい，と結論づけることができる．

この12次方程式を得るには，\(x\) を表す式から \(y\) を消去し，さらにすべての平方根を除去すればよい．あるいは，まず次の根号を含まない方程式に立ち返ることもできる： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2} = z^{2} \left(x + \frac{m y - p}{2 z^{2}}\right)^{2}. \] ここで \(z^{2}\) にその値 \[2y + \frac{m^{2}}{4} - n\] を代入した後，\(x\,\)[*1]を消去すればよい．

議論をより一般化するため，\(z^{2}\) の値を単に\(2y + \frac{m^{2}}{4} - n\)とするのではなく， \[ k \left(2y + \frac{m^{2}}{4} - n\right) \] と置くことにしよう．定数 \(k\) は，\(y\) を消去した後に現れる \(x\) の方程式の次数に影響を及ぼさないことは明白である．このようにすれば，次の式が得られる： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2} = k \left(2y + \frac{m^{2}}{4} - n\right) x^{2} + (m y - p) x + \frac{(m y - p)^{2}}{4 k \left(2y + \dfrac{m^{2}}{4} - n\right)}. \] これは，\(y\) についての方程式 \[\frac{(m y - p)^{2}}{4} = \left(2y + \frac{m^{2}}{4} - n\right)(y^{2} - q)\] を用いると，次のように書き換えられる： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2}=k \left(2y + \frac{m^{2}}{4} - n\right) x^{2} + (m y - p) x + \frac{y^{2} - q}{k}. \] すなわち， \[ x^{4} + m x^{3} + \left[k n + (1 - k)\left(2y + \frac{m^{2}}{4}\right)\right] x^{2} + p x + \frac{q + (k - 1) y^{2}}{k} = 0. \] ここで簡略のために \[ x^{4} + m x^{3} + n x^{2} + p x + q = {X}, \] そして\(k - 1 = h\)とおけば，この方程式は次のように書ける： \[ {X} + h \left[ \left(n - \frac{m^{2}}{4} - 2y\right) x^{2} + \frac{y^{2} - q}{k} \right] = 0. \] あとは，方程式 \[ y^{3} - \frac{n}{2} y^{2} + \frac{m p - 4 q}{4} y + \frac{(4n - m^{2}) q - p^{2}}{8} = 0 \] を用いて，\(y\) を消去すればよい． この3次方程式の3つの根を \(y', y'', y'''\) と名づけると，未知数 \(y\) を消去して得られる \(x\) についての方程式は，次の3つの式の積として表される（第13項）： \[ \begin{aligned} &X + h\left[\left(n - \frac{m^{2}}{4} - 2y'\right)x^{2} + \frac{y'^{2} - q}{k}\right], \\ &X + h\left[\left(n - \frac{m^{2}}{4} - 2y''\right)x^{2} + \frac{y''^{2} - q}{k}\right], \\ &X + h\left[\left(n - \frac{m^{2}}{4} - 2y'''\right)x^{2} + \frac{y'''^{2} - q}{k}\right]. \end{aligned} \] この積を 0 に等しいとする． 簡略のため次のように置く： \[ \begin{aligned} A &= X + h\left[\left(n - \frac{m^{2}}{4}\right)x^{2} - \frac{q}{k}\right], \\ B &= -2h x^{2}, \\ C &= \frac{h}{k}. \end{aligned} \] また，次のように置く： \[ \begin{aligned} \alpha &= y' + y'' + y''', \\ \beta &= y'y'' + y'y''' + y''y''', \\ \gamma &= y'^{2} + y''^{2} + y'''^{2}, \\ \delta &= y'y''y''', \\ \varepsilon &= y'^{2}y''^{2} + y'^{2}y'''^{2} + y''^{2}y'''^{2}. \end{aligned} \] すると，この3つの式の積は次のようになる： \[ A^{3} + A^{2}B\alpha + A^{2}C\gamma + AB^{2}\beta + ABC(\alpha\beta - 3\delta)+ AC^{2}\varepsilon + B^{3}\delta + B^{2}C\alpha\delta + BC^{2}\beta\delta + C^{3}\delta^{2}. \] さらに簡略のため，次のように置く： \[ a = \frac{n}{2}, \quad b = \frac{mp - 4q}{4}, \quad c = \frac{p^{2} - (4n - m^{2})q}{8}. \] このとき，\(y\) についての方程式は \[ y^{3} - a y^{2} + b y - c = 0 \] と書ける．そして，よく知られているように，方程式の性質から， \[ \alpha = a, \quad \beta = b, \quad \delta = c \] が得られ，そこから次が導かれる： \[ \gamma = a^{2} - 2b, \quad \varepsilon = b^{2} - 2ac. \] したがって，次の2つの式 \begin{align*} &A + B y + C y^{2} = 0,\\ &y^{3} - a y^{2} + b y - c = 0 \end{align*} から \(y\) を消去して得られる \(x\) についての方程式は： \[ \begin{aligned} A^{3} &+ a A^{2} B + (a^{2} - 2b) A^{2} C + b A B^{2} + (ab - 3c) A B C \\ &+ (b^{2} - 2ac) A C^{2} + c B^{3} + ac B^{2} C + bc B C^{2} + c^{2} C^{3} = 0. \end{aligned} \] ここで \(A, B, C\) および \(a, b, c\) の値を代入すれば，\(x\) についての12次方程式が得られる．なぜなら \(A\) は \(x\) の4次までの項を含み，\(B\) は \(x^{2}\) の項を含み，他は \(x\) を含まないからである．

したがって，この12次方程式の根は，上記と同様の形になる： \[ x = \frac{1}{2} \left(z - \frac{m}{2} \pm \sqrt{z^{2} - m z + \frac{m^{2}}{4} - 4y + \frac{2my - p}{z}}\right).\,\,^{\text{[*2]}} \] ここでは， \[ z = \pm \sqrt{2y + \frac{m^{2}}{4} - nk} \] と置いている． そして，ここに余計な根が混入することはない．なぜなら，\(y\) の3つの値と，2つの平方根の複号を組み合わせることで，問題となっている方程式のちょうど12個の根が得られるからである．

第27項の場合については，\(k = 1\) とすればよい．このとき，\(h = 0\) であり， \[ A = X = x^{4} + m x^{3} + n x^{2} + p x + q, \quad B = 0, \quad C = 0. \] したがって，方程式は \(A^{3} = 0\)，すなわち， \[ \left(x^{4} + m x^{3} + n x^{2} + p x + q\right)^{3} = 0 \] となり，これは与えられた方程式を3乗したものである．したがって，根はもとの方程式と同じだが，各根が3重根として現れる．

このことから，我々は4次方程式の根を与える式が実際には12個の根を含みながら，それぞれの根が他の2つの根と等しいことから，結局4つの根に帰着する理由を明確に理解できる． さらに，先ほどの証明によれば，この根の重複は平方根の複号によるものではなく，\(y\) の消去によって \(X\) が3乗されることに起因する．したがって，\(x\) の式にどの \(y\) の値を代入しても，得られる根は常に同じ4つである．

  1. 消去するのは\(y\)であろう．
  2. \(2my-p\)は\(2(my-p)\)の誤り．

29．この問題をさらに明確にするために，次の点に注目しよう．すなわち，\(y\) についての簡約方程式が成立する場合（これは \(y\) に3つの根があるため，3通りの異なる仕方で起こりうる），第27項で示したように，与えられた方程式は次の形に書き換えることができる： \[ \left(x^{2} + \frac{m x}{2} + y\right)^{2} - z^{2} \left(x + \frac{m y - p}{2 z^{2}}\right)^{2} = 0. \] これは，次の2式の積に他ならない： \[ \begin{aligned} x^{2} + \frac{m x}{2} + y + z \left(x + \frac{m y - p}{2 z^{2}}\right) &= 0, \\ x^{2} + \frac{m x}{2} + y - z \left(x + \frac{m y - p}{2 z^{2}}\right) &= 0, \end{aligned} \] すなわち，次の2つの式である： \[ \begin{aligned} x^{2} + \left(\frac{m}{2} + z\right)x + y + \frac{m y - p}{2 z} &= 0, \\ x^{2} + \left(\frac{m}{2} - z\right)x + y - \frac{m y - p}{2 z} &= 0. \end{aligned} \] したがって，\(y\) にどの根を代入しても，これらの方程式の根は常に与えられた方程式の4つの根と一致することになる．

ここで，与えられた方程式の4つの根を \(a, b, c, d\) と名づけることにしよう．すると，これらのうち2つの根は上の一方の方程式に含まれ，残りの2つは他方に含まれるはずである．したがって，方程式の性質から次が得られる： \[ \begin{alignedat}{2} a + b &= -\frac{m}{2} - z, \qquad &ab &= y + \frac{m y - p}{2 z}, \\ c + d &= -\frac{m}{2} + z, \qquad &cd &= y - \frac{m y - p}{2 z}. \end{alignedat} \] ここから次の関係が導かれる： \[ z = \frac{c + d - a - b}{2}, \quad y = \frac{ab + cd}{2}. \] この \(y\) の値から，なぜ \(y\) についての簡約方程式が3次となるのか，その理由を直ちに理解することができる． 実際，根 \(a, b, c, d\) に可能な限りのあらゆる可能な置換を施したとき，\(\frac{ab + cd}{2}\) がとり得る値の数だけ \(y\) が異なる値をとることは明らかである．ところが，そのようにして得られる値は，次の3つに限られる： \[ \frac{ab + cd}{2}, \quad \frac{ac + bd}{2}, \quad \frac{ad + cb}{2}. \] したがって，\(y\) についての方程式は，これら3つの値を根として含まなくてはならず，ゆえに3次方程式となるのである．

30．以上の考察から，4次方程式の簡約方程式を直接導く方法，ひいては4次方程式の一般的な解法を得る方法が導かれる． というのも，4つの根 \(a, b, c, d\) に対して，組合せ \(ab + cd\) は次の3通りの変化しかとらない： \[ ab + cd, \quad ac + bd, \quad ad + cb. \] したがって， \[ab + cd = u\] とおけば，\(u\) についての3次方程式が得られ，その根は \[ ab + cd, \quad ac + bd, \quad ad + bc \] であり，次の形をしている： \[ u^{3} - A u^{2} + B u - C = 0. \] ここで，方程式の性質から次が成り立つ： \[ \begin{aligned} A &= ab + cd + ac + bd + ad + cb, \\ B &= (ab + cd)(ac + bd) + (ab + cd)(ad + cb) + (ac + bd)(ad + cb), \\ C &= (ab + cd)(ac + bd)(ad + cb). \end{aligned} \] すなわち， \[ \begin{aligned} A &= ab + ac + ad + bc + bd + cd, \\ B &= a^{2}(bc + bd + cd) + b^{2}(ac + ad + cd) + c^{2}(ab + ad + bd) + d^{2}(ab + ac + bc), \\ C &= abcd(a^{2} + b^{2} + c^{2} + d^{2}) + a^{2}b^{2}c^{2} + a^{2}b^{2}d^{2} + a^{2}c^{2}d^{2} + b^{2}c^{2}d^{2}. \end{aligned} \] さて，これら \(A, B, C\) の値は，もとの方程式の係数 \(m, n, p, q\) によって，冪根の抽出を一切伴わずに与えられることが容易にわかる．なぜなら，この方程式の根 \(a, b, c, d\) にどのような置換を施しても，それらの値は変わらないからである．したがって，それぞれの値はただ1つの同じ値しか取りえない． 実際， \[ \begin{aligned} -m &= a + b + c + d, \\ n &= ab + ac + ad + bc + bd + cd, \\ -p &= abc + abd + acd + bcd, \\ q &= abcd. \end{aligned} \] という関係があるので，まず， \[A = n.\] 次に \(B\) を求めるには，たとえば， \[ a(bc + bd + cd) = -p - bcd, \] 同様に \[ b(ac + ad + cd) = -p - acd, \] などが成り立つので，これらを用いると： \[ B = (a + b + c + d)(-p) - 4 abcd, \] すなわち， \[B = mp - 4q.\] 最後に，\(C\)を得るためには，次のことに注意する．すなわち， \[ a^{2} + b^{2} + c^{2} + d^{2} = m^{2} - 2n \] であるから，\(abcd(a^{2} + b^{2} + c^{2} + d^{2})\) の部分は \((m^{2} - 2n)q\) となる． そして，残りの部分を得るために，\(p\) の平方をとり次の式を導く． \begin{align*} a^{2}b^{2}c^{2} + a^{2}b^{2}d^{2} + a^{2}c^{2}d^{2} + b^{2}c^{2}d^{2} &= p^{2} - 2 abcd(ab + ac + bc + ad + bd + cd)\\ &= p^{2} - 2nq. \end{align*} したがって， \[ C = (m^{2} - 4n)q + p^{2}. \] 以上により，簡約方程式は次のように書ける： \[ u^{3} - n u^{2} + (mp - 4q) u - (m^{2} - 4n)q - p^{2} = 0. \] \(u = 2y\) と置けば，第27項で得られた \(y\) についての簡約方程式と一致する．

31．さて，\(u\) の値の1つがわかっているときに，いかにして与えられた方程式の4つの根 \(a, b, c, d\) が求められるかを見てみよう． \[u = ab + cd\quad \text{および}\quad abcd = q\] であることから，\(ab\) と \(cd\) は次の2次方程式の根であることは明らかである： \[ t^{2} - ut + q = 0. \] この2つの根を \(t'\), \(t''\) と名づければ，それぞれ， \[ab = t', \quad cd = t''.\] さらに，次の関係がある： \[ -p = ab(c + d) + cd(a + b) = t'(c + d) + t''(a + b). \] また， \[a + b + c + d = -m\] であるから，次のように書ける： \[ a + b = \frac{p - m t'}{t' - t''}, \quad c + d = \frac{p - m t''}{t'' - t'}. \] したがって， \[ab = t'\quad\text{および}\quad cd = t''\] であることから，\(a\) と \(b\) は次の2次方程式の根であることは明らかである： \[x^{2} - \frac{p - m t'}{t' - t''} x + t' = 0,\] また，\(c\) と \(d\) は次の方程式の根である： \[x^{2} - \frac{p - m t''}{t'' - t'} x + t'' = 0.\] このことから，\(u\) についての簡約方程式の根のいずれか1つが既知となれば，もとの方程式の4つの根 \(a, b, c, d\) を求めるのに十分であることが判明した． また，この簡約方程式のどの根を採択したとしても，常に同じ4つの根 \(a, b, c, d\) が得られる．というのも，もし \(u = ab + cd\) の代わりに \(u = ac + bd\) や \(u = ad + bc\) を用いたとしても，式の中で \(b\) が \(c\) または \(d\) と入れ替わるに過ぎないからである．

32．4次方程式をより簡単に解く方法として， \[ z = \frac{c + d - a - b}{2} \quad \text{あるいは} \quad s = c + d - a - b \] を根としてもつ（第29項）簡約方程式を用いることができる（簡単のため \(s = 2z\) と置いた）． まず，この簡約方程式が何次でどのような形になるかを知るには，\(c + d - a - b\) に \(a, b, c, d\) のあらゆる可能な置換を施せばよい．すると，次の6通りが得られる： \[ \begin{aligned} &a + b - c - d, \\ &a + c - b - d, \\ &a + d - c - b , \\ &c + d - a - b, \\ &b + d - a - c, \\ &b + c - a - d. \end{aligned} \] これらが \(s\) についての簡約方程式の根となるので，この方程式は6次である． しかし，これら6つの量は2つずつ符号が逆の対になるため，この方程式には \(s\) の偶数次の項しか現れない．

したがって，\(s^{2} = t\) とおけば，\(t\) について3次の簡約方程式が得られ，その根は次の3つ： \[ (a + b - c - d)^{2}, \quad (a + c - b - d)^{2}, \quad (a + d - b - c)^{2}. \] この方程式は，先に \(u\) に関して行ったようにして（第30項），求めることができる．しかしながら，この目的のためにあらためて計算に着手する必要はなく，\(a + b - c - d\) の平方が次のようになることに注目すれば十分である． \[ a^{2} + b^{2} + c^{2} + d^{2} + 2ab + 2cd - 2ac - 2ad - 2bc - 2bd. \] ここで， \begin{gather*} ab + ac + ad + bc + bd + cd = n, \\ a^{2} + b^{2} + c^{2} + d^{2} = m^{2} - 2n \end{gather*} であるから， \[ (a + b - c - d)^{2} = s^{2} = t \] より， \[ t = m^{2} - 4n + 4(ab + cd), \] すなわち， \[ t = m^{2} - 4n + 4u. \] したがって，\(t\) についての求める簡約方程式を得るには，\(u\) についての簡約方程式（第30項）において \(u = \frac{t - m^{2} + 4n}{4}\) と置き換えればよく，次のようになる． \[ t^{3} - (3m^{2} - 8n)t^{2} + (3m^{4} - 16m^{2}n + 16n^{2} + 16mp - 64q)t - (m^{3} - 4mn + 8p)^{2} = 0. \] さて，この方程式の3つの根を \(t',\ t'',\ t'''\) とすると, \[ (a + b - c - d)^2 = t', \quad (a + c - b - d)^2 = t'', \quad (a + d - b - c)^2 = t''' \] となる（仮定より）． したがって，平方根をとると： \[ a + b - c - d = \sqrt{t'}, \quad a + c - b - d = \sqrt{t''}, \quad a + d - b - c = \sqrt{t'''}. \] これらと \[a + b + c + d = -m\] を組み合わせれば，\(a, b, c, d\) の値が次のように得られる： \[ \begin{aligned} a &= \frac{-m + \sqrt{t'} + \sqrt{t''} + \sqrt{t'''}}{4}, \\[+3pt] b &= \frac{-m + \sqrt{t'} - \sqrt{t''} - \sqrt{t'''}}{4}, \\[+3pt] c &= \frac{-m - \sqrt{t'} + \sqrt{t''} - \sqrt{t'''}}{4}, \\[+3pt] d &= \frac{-m - \sqrt{t'} - \sqrt{t''} + \sqrt{t'''}}{4}. \end{aligned} \] このようにして，\(t\) についての簡約方程式以外のいかなる方程式も解くことなく，もとの方程式の4つの根を得ることができる． しかしここで1つの困難が生じる． それは，それぞれの平方根 \(\sqrt{t'},\ \sqrt{t''},\ \sqrt{t'''}\) は正負いずれの符号でも取りうるため， 先に与えた式には次のような4つの値も含まれてしまうことである： \begin{align*} \frac{-m + \sqrt{t'} + \sqrt{t''} - \sqrt{t'''}}{4}, \\ \frac{-m + \sqrt{t'} - \sqrt{t''} + \sqrt{t'''}}{4}, \\ \frac{-m - \sqrt{t'} + \sqrt{t''} + \sqrt{t'''}}{4}, \\ \frac{-m - \sqrt{t'} - \sqrt{t''} - \sqrt{t'''}}{4}. \end{align*} これらは \(a, b, c, d\) の値そのものではなく，それらの和 \[a + b + c + d = -m\]に対する補数[*1]である．

4mn + 8p)^{2}, \] したがって， \[ m^{3} - 4mn + 8p = \sqrt{t'} \sqrt{t''} \sqrt{t'''}\,\,^{\text{(*)}}. \] ここで，\(\theta',\ \theta'',\ \theta'''\) をそれぞれ \(\sqrt{t'},\ \sqrt{t''},\ \sqrt{t'''}\) の正の値とすれば， \[ \sqrt{t'} = \pm \theta', \quad \sqrt{t''} = \pm \theta'', \quad \sqrt{t'''} = \pm \theta'''. \] このとき，\(m^{3} - 4mn + 8p\) が正であれば， \[\sqrt{t'} = \theta' \quad\text{と}\quad \sqrt{t''} = \pm \theta'',\quad \sqrt{t'''} = \pm \theta''',\] または \[\sqrt{t''} = \theta'' \quad\text{と}\quad \sqrt{t'} = \pm \theta',\quad\sqrt{t'''} = \pm \theta''',\] または \[\sqrt{t'''} = \theta''' \quad\text{と}\quad \sqrt{t'} = \pm \theta',\quad \sqrt{t''} = \pm \theta''\] と選ぶことができる． このとき，複号は，\(\theta''\) と \(\theta'''\)，または \(\theta'\) と \(\theta'''\)，あるいは \(\theta'\) と \(\theta''\) の間で一致させる． この場合，もとの方程式の4つの根は次のように与えられる： \[ \begin{aligned} &\frac{-m + \theta' + \theta'' + \theta'''}{4}, \\[+3pt] &\frac{-m + \theta' - \theta'' - \theta'''}{4}, \\[+3pt] &\frac{-m - \theta' + \theta'' - \theta'''}{4}, \\[+3pt] &\frac{-m - \theta' - \theta'' + \theta'''}{4}. \end{aligned} \] 逆に，\(m^{3} - 4mn + 8p\) が負であれば， \[ \sqrt{t'} = -\theta' \quad\text{と}\quad \sqrt{t''} = \pm \theta'',\quad \sqrt{t'''} = \pm \theta''', \] または \[ \sqrt{t''} = -\theta'' \quad\text{と}\quad \sqrt{t'} = \pm \theta',\quad \sqrt{t'''} = \pm \theta''', \] または \[ \sqrt{t'''} = -\theta''' \quad\text{と}\quad \sqrt{t'} = \pm \theta',\quad \sqrt{t''} = \pm \theta'' \] と選ぶことができる． この場合，もとの方程式の4つの根は次のように与えられる： \[ \begin{aligned} &\frac{-m + \theta' + \theta'' - \theta'''}{4}, \\[+3pt] &\frac{-m + \theta' - \theta'' + \theta'''}{4}, \\[+3pt] &\frac{-m - \theta' + \theta'' + \theta'''}{4}, \\[+3pt] &\frac{-m - \theta' - \theta'' - \theta'''}{4}. \end{aligned} \]

33．これまでに見てきたフェラーリの方法は，4次方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] を，次の2つの2次方程式に分解することへと導いた（第29項）： \[ \begin{aligned} x^{2} + \left( \frac{m}{2} + z \right)x + y + \frac{my - p}{2z} &= 0, \\[+3pt] x^{2} + \left( \frac{m}{2} - z \right)x + y - \frac{my - p}{2z} &= 0. \end{aligned} \] これらを解くことで，もとの方程式の4つの根を得ることができることは，すでに見たとおりである． この分解は，より簡単かつ直接的な方法でも得ることができる．すなわち，まず，もとの方程式が次のような2つの2次方程式の積であると仮定する： \[ x^{2} + f x + g = 0, \quad x^{2} + h x + k = 0. \] そして，対応する項を比較することで，係数 \(f, g, h, k\) を定めるのである． これはデカルトが行った方法であり，彼がその創始者と見なされている未定係数法の起源となったものである． 実際，この2つの方程式を掛け合わせると，次のようになる： \[ x^{4} + (f + h)x^{3} + (fh + g + k)x^{2} + (fk + gh)x + gk = 0. \] これを，もとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] と項ごとに比較すれば，次の4つの方程式が得られる： \[ f + h = m, \quad fh + g + k = n, \quad fk + gh = p, \quad gk = q. \] これらの式によって，未知数 \(f, g, h, k\) を定めることができる．

34．まずデカルトに従って，もとの方程式の第2項が消えていると仮定しよう．すなわち，\(m = 0\) とする． このとき \(f + h = 0\)，したがって \(h = -f\) となるから， この場合，もとの方程式は次のような2つの2次方程式の積として得られることになる： \[ x^{2} + f x + g = 0, \quad x^{2} - f x + k = 0. \] このとき，係数 \(f, g, k\) を定めるために，次の3つの方程式が得られる： \[ g + k - f^{2} = n, \quad (k - g)f = p, \quad gk = q. \] 最初の2つの式から， \[ g = \frac{n + f^{2} - \dfrac{p}{f}}{2}, \quad k = \frac{n + f^{2} + \dfrac{p}{f}}{2} \] が得られ，これらの値を最後の式に代入すると， \[ \left(n + f^{2}\right)^{2} - \frac{p^{2}}{f^{2}} = 4q \] となる． 両辺に \(f^{2}\) を掛けて整理すれば，\(f\) についての6次方程式が得られる： \[ f^{6} + 2n f^{4} + \left(n^{2} - 4q\right)f^{2} - p^{2} = 0. \] これは6次方程式ではあるが，未知数の偶数次の項のみを含んでいるため， 3次方程式と同様の方法で解くことができる．

これが，4次方程式に対するデカルトの方法である． もっとも，この著者は，構成される2次方程式を最初から次のように表している（これは上で求めた \(g\) および \(k\) の値を代入したものである）： \[ x^{2} + f x + \frac{f^{2}}{2} + \frac{n}{2} - \frac{p}{2f} = 0, \quad x^{2} - f x + \frac{f^{2}}{2} + \frac{n}{2} + \frac{p}{2f} = 0. \] しかし，彼は，我々がここで示したような分析的な手続きによってこれらの式を得たと考えるのが自然であろう． このことは，スコーテンの『注解書（Commentaire）』や，フッデによる書簡『方程式の簡約法（la réduction des équations）』からも見て取れる．

35．上述の解法は，第26項以降で述べた解法と本質的に同じものであり， \(m = 0\) のとき，それぞれの解法における未知数 \(f\) と \(z\) が同じ量を表していることは明らかである． したがって，フェラーリの解法について述べた主要な注意点は， デカルトの解法にもそのまま適用でき，ここで改めて詳述する必要はない． しかし，このデカルトの解法の原理そのものを個別に検討し， そこから導かれる結果をアプリオリに導出しておくことは有益である．

この原理は，すでに見たように，もとの方程式が次のような2次方程式で割り切れると仮定することにある： \[ x^{2} + f x + g = 0. \] すなわち，もとの方程式がこの2次方程式と2つの共通根をもつということである． したがって，この条件を満たすための必要条件は，第12項の方法によって求めることができる． 実際，5項式 \[ x^{4} + m x^{3} + n x^{2} + p x + q \] を3項式 \[x^{2} + f x + g\] で割ると，商は， \[ x^{2} + (m - f)x + n - g - f(m - f), \] 余りは， \[ \bigl[p - g(m - f) - f\left[n - g - f(m - f)\right]\bigr]x + q - g\left[n - g - f(m - f)\right]. \] したがって，割り切れるためには，この余りが \(x\) に依存せずに 0 でなければならず， 次の2つの方程式が得られる： \[ \begin{aligned} &p - g(m - f) - f\left[n - g - f(m - f)\right] = 0, \\ &q - g\left[n - g - f(m - f)\right] = 0. \end{aligned} \] これらによって \(f\) および \(g\) を定めることができる． 第1式から，まず \(g\) の値は \[ g = \frac{p - n f + m f^{2} - f^{3}}{m - 2f}. \] この値を第2式に代入すると， \[ q - (n - m f + f^{2}) \frac{p - n f + m f^{2} - f^{3}}{m - 2f} + \frac{(p - n f + m f^{2} - f^{3})^{2}}{(m - 2f)^{2}} = 0, \] すなわち， \[ \left(f^{3} - m f^{2} + n f - p\right)^{2} - \left(f^{3} - m f^{2} + n f - p\right)\left(f^{2} - m f + n\right)(2f - m) + q(2f - m)^{2} = 0. \] これを \(f\) について整理すれば，次の6次方程式が得られる： \[ \begin{aligned} f^{6} - 3m f^{5} + (3m^{2} + 2n)f^{4}- &m(m^{2} + 4n)f^{3}+ (2m^{2}n + m p + n^{2} - 4q)f^{2} \\ &\qquad - m(mp + n^{2} - 4q)f + m n p - m^{2} q - p^{2} = 0. \end{aligned} \] これは第33項で得られた4つの条件式から導かれる方程式と一致する．

この方程式は，見てのとおり6次式であり，すべての次数の項を含んでいる． しかし，第34項ですでに検討したように，\(m = 0\) とすれば，未知数の奇数次の項はすべて消えるので， この方程式は3次方程式と同様の方法で解くことができるようになる． だが，未知数の奇数次の項を消すために \(m = 0\) を仮定する必要すらない． よく知られている方法によって，第2項を消去すれば十分で， \[ f = l + \frac{3m}{6} = l + \frac{m}{2} \] とおけば，その他の奇数次の項も同時に消えることがわかる． このとき，\(l\) についての方程式は \(l^{2}\) の冪のみを含み，次のようになる： \[ l^{6} - \left( \frac{3m^{2}}{4} - 2n \right) l^{4} + \left( \frac{3m^{4}}{16} - m^{2}n + m p + n^{2} - 4q \right) l^{2} - \left( \frac{m^{3}}{8} - \frac{m n}{2} + p \right)^{2} = 0. \] \(t = 4l^{2}\)とすれば，この方程式は第32項に現れた \(t\) についての簡約方程式と一致することがわかる． また，そこでは，\(t = s^{2}\)，\(s = 2z\) としていたので，\(l = z\) となる． したがって，これまでの式に現れる \(f - \frac{m}{2}\) という量は， 第27項以降に現れた \(z\) と一致する．これは \(m = 0\) を仮定することなく成り立つ． このことにより，これまでに見てきたさまざまな解法の関係がより明確になるのである．

36．上述のように，デカルトの方法によって得られる6次の簡約方程式では，第2項を消去すると未知数の奇数次の項もすべて消える．ここでは，その理由を検討しよう．そのために，次の点に注目する： 2次方程式 \(x^{2} + f x + g = 0\) は，もとの方程式の因数なので，もとの方程式の根を \(a, b, c, d\) とすれば， この2次方程式は，これら4つの根のうち任意の2つを含んでいなければならない． したがって，次のような6通りの組み合わせが考えられる： \[-f = a + b,\quad g = ab,\] または \[-f = a + c,\quad g = ac,\] または \[-f = a + d,\quad g = ad,\] または \[-f = b + c,\quad g = bc,\] または \[-f = b + d,\quad g = bd,\] そして最後に \[-f = c + d,\quad g = cd.\] したがって，\(f\) についての方程式も \(g\) についての方程式も6次であることがわかる． すなわち，次数は4つのものから2つを選ぶ組合せの数に等しく， これはよく知られているように \(\frac{4\cdot 3}{2} = 6\) である． したがって，この方法によって，\(f\) についての方程式と \(g\) についての方程式の両方を，すでに何度か行ってきたようにして，それぞれの係数の値を求めることから直接導くことができる． まず，\(f\) についての方程式を次の一般形で表す： \[ f^{6} + A f^{5} + B f^{4} + C f^{3} + D f^{2} + E f + F = 0. \] この方程式の根は， \[ -a - b,\quad -a - c,\quad -a - d,\quad -b - c,\quad -b - d,\quad -c - d \] であるから， \[ \begin{aligned} {A} &= a + b + a + c + a + d + b + c + b + d + c + d \\ &= 3(a + b + c + d) = -3m, \\[+5pt] {B} &= (a + b)(a + c + a + d + b + c + b + d + c + d) \\ &\quad + (a + c)(a + d + b + c + b + d + c + d) \\ &\quad + (a + d)(b + c + b + d + c + d) \\ &\quad + (b + c)(b + d + c + d) \\ &\quad + (b + d)(c + d) \\ &= 3(a^{2} + b^{2} + c^{2} + d^{2}) + 8(ab + ac + ad + bc + bd + cd) \\ &= 3(m^{2} - 2n) + 8n = 3m^{2} + 2n, \end{aligned} \] 等々以下〔\(C, D, \dots\)についても〕同様．

さて，\(f\) についてのこの方程式の第2項を消したいとすれば， よく知られた規則に従って，すべての根に \(\frac{A}{6}\) を加える必要がある． すなわち，\(f\) の代わりに \(l - \frac{A}{6}\) を代入するということであり， これによって \(l\) についての変換方程式が得られる． \[A = 3(a + b + c + d)\] だから， \[ l = f + \frac{A}{6} = f + \frac{a + b + c + d}{2}. \] したがって，この変換方程式の根は次のようになる： \[ \begin{aligned} &\frac{a + b + c + d}{2} - a - b, \\ &\frac{a + b + c + d}{2} - a - c, \\ &\dots\dots\dots\dots\dots\dots\dots, \end{aligned} \] すなわち， \[ \begin{aligned} &\frac{a + b - c - d}{2}, \\ &\frac{a + c - b - d}{2}, \\ &\frac{a + d - b - c}{2}, \\ &\frac{b + c - a - d}{2}, \\ &\frac{b + d - a - c}{2}, \\ &\frac{c + d - a - b}{2}. \end{aligned} \] ここで，各根はその符号を反転させたものと対になっていることがわかる． したがって，これらの平方をとり，\(l^{2}\) を未知数とみなせば， 異なる値は次の3つに限られる： \begin{align*} \left( \frac{a + b - c - d}{2} \right)^{2}, \\ \left( \frac{a + c - b - d}{2} \right)^{2}, \\ \left( \frac{a + d - b - c}{2} \right)^{2}. \end{align*} このことから，\(l^{2}\) について整理した方程式は3次式となる． すなわち，\(l\) についての方程式は6次で，未知数の偶数次の項のみを含むことになる． これはすでに別の方法で確認したとおりであり， 4次方程式が解けるのはこの幸運な事情によるものである． また，このことから第35項で \(l\) についての方程式が \(z\) についての方程式と一致していた理由も明らかになる． 実際，\(z\) の値はここで得られた \(l\) の値と一致している（第32項）．

37．さらに次のことも指摘できる．すなわち， \[-m = a + b + c + d\] であり， \(f\) の値が， \[-a - b,\ -a - c,\ -a - d,\ -b - c,\ -b - d,\ -c - d\] であることから， \(m - f\) もこれらと同一の値になる． したがって，\(f\) についての方程式は，\(f\) を \(m - f\) に置き換えても変わらない形でなければならない． すなわち，\(f = \frac{m}{2} + l\) とおけば，\(m - f = \frac{m}{2} - l\) となるので， \(l\) についての変換方程式は，\(l\) を \(-l\) に置き換えても変わらない形， すなわち \(l\) の偶数次の項のみを含む形でなければならない．

さて，第35項の \(g\) の式にこの \(f\) の値を代入すると， \[ g = \frac{l^{2}}{2} + \frac{m l}{4} + \frac{4n - m^{2}}{8} + \frac{4nm - m^{3} - 8p}{16l}. \] また，方程式 \[x^{4} + m x^{3} + n x^{2} + p x + q = 0\] を分解するために用いた2つの因子 \[ x^{2} + f x + g = 0,\quad x^{2} + (m - f)x + n - g - f(m - f) = 0 \] は，それぞれ次のようになる： \[ \begin{aligned} x^{2} + \left( \frac{m}{2} + l \right)x + \frac{l^{2}}{2} + \frac{m l}{4} + \frac{4n - m^{2}}{8} + \frac{4nm - m^{3} - 8p}{16l} = 0, \\[+3pt] x^{2} + \left( \frac{m}{2} - l \right)x + \frac{l^{2}}{2} - \frac{m l}{4} + \frac{4n - m^{2}}{8} - \frac{4nm - m^{3} - 8p}{16l} = 0. \end{aligned} \] これらは第29項で得られた方程式と一致していることがわかる．そのためには，上の式で \(l = z\) とし， 次の式（第28項） \[ y = \frac{4z^{2} + 4n - m^{2}}{8} \] を用いて第29項の式の \(y\) を \(z\) で表せばよい．

38．まず明らかなのは，チルンハウス氏の方法によって4次方程式を解くには， 3次方程式の場合のようにすべての中間項を消去する必要はなく， 未知数が奇数次で現れる第2項と第4項を消すだけで十分であるということである． このとき，方程式は2次方程式と同様の方法で解ける形になる． そのために，3次方程式の場合（第10項）と同様に，補助方程式 \[ x^{2} = b x + a + y \] を用いる．ここで \(a\) と \(b\) は2つの未定係数である． この補助方程式を用いて，もとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] から未知数 \(x\) を消去すると，\(y\) についての4次の変換方程式が得られる（第14項）． この変換方程式では，\(y^{3}\) の係数は \(a\) および \(b\) の1次の関数， \(y^{2}\) の係数は \(a\) および \(b\) の2次の関数， \(y\) の係数は \(a\) および \(b\) の3次の関数， …というように構成される． したがって，第2項と第4項を同時に消すには，\(a\) および \(b\) が 1次と3次の2つの方程式を満たすように定められなければならず， その結果，\(a\) または \(b\) についての3次の簡約方程式が得られる． このことから，チルンハウス氏の方法も4次方程式に対して有効であると結論づけられる． 以下，このことを実際の計算によって確かめることにしよう．

すでに述べたとおり，第2項と第4項を消すことにする．そのための条件は： \[ \begin{aligned} &4g - A = 0, \\ &4g^{3} - 3 A g^{2} + 2 B g - C = 0. \end{aligned} \] 第1式から \[g = \frac{A}{4}\] を得て，これを第2式に代入すると： \[ A^{3} - 4 A B + 8 C = 0. \] この式に \(A, B, C\) の値を代入すれば，\(f\) についての3次方程式が得られる． 整理すると： \[ \begin{aligned} &(m^{3} - 4 m n + 8 p) f^{3} - (3 m^{4} - 14 m^{2} n + 8 n^{2} + 2m p - 32 q) f^{2} \\ &\qquad+ [m^{5} - 16 m^{3} n + 20 m^{2} p + 16 m(n^{2} - 2q) - 16 n p] f \\ &\qquad\qquad- m^{6} + 6 m^{4} n - 8 m^{3} n p - 8 m^{2}(n^{2} - q) + 8 m n^{2} p - 8 p^{2} = 0.\,\,^{\text{[*2]}} \end{aligned} \] この方程式により \(f\) を決めれば，\(y\) についての方程式は，\(g = \frac{A}{4}\) であることから，次のようになる． \[ y^{4} - \left( \frac{3 A^{2}}{8} - B \right) y^{2} - \frac{3 A^{4}}{256} + \frac{A^{2} B}{16} - \frac{A C}{4} + D = 0. \] さらに \(C\)にその値 \(\frac{A B}{2} - \frac{A^{3}}{8}\) を代入すれば： \[ y^{4} - \left( \frac{3 A^{2}}{8} - B \right) y^{2} + \frac{5 A^{4}}{256} - \frac{A^{2} B}{16} + D = 0. \] これは2次方程式と同様の方法で解くことができる． したがって，\(f\) および \(y\) が求まり，それにより \(x\) は次の式で与えられる： \[ x = \frac{ q - (n - m f + f^{2}) \left( \dfrac{A}{4} + y \right) + \left( \dfrac{A}{4} + y \right)^{2} }{ f^{3} - m f^{2} + n f - p + (m - 2f) \left( \dfrac{A}{4} + y \right) }. \] 先の方程式から求まる \(y\) の4つの値から，もとの方程式の4つの根がことごとく導かれる． どの \(f\) の根を採用しても，得られる \(x\) の値は同一である． このことは，必要であれば，第28項で用いたのと同様の方法で立証可能である．

この場合，消去すべき項は第2項，第3項，第4項の3つであり， それには3つの未定係数を含む補助方程式が必要となる．その形は次のようになる： \[ x^{3} + f x^{2} + g x + h + y = 0. \] この補助方程式と，もとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] を用いて \(x\) を消去すれば，\(y\) についての4次の変換方程式が得られる．その形は： \[ y^{4} + A y^{3} + B y^{2} + C y + D = 0. \] ここで，2項のみからなる方程式 \[y^{4} + D = 0\] を得るには， \(A = 0,\ B = 0,\ C = 0\) を仮定しなければならない． ところで第14項で一般的に示したように， \(A\) は \(f, g, h\) の1次の関数，\(B\) は2次の関数，\(C\) は3次の関数となる． したがって，\(f, g, h\) を決定するには次の3つの方程式を解く必要がある： \[ A = 0,\quad B = 0,\quad C = 0. \] このうち，第1式の次数は1次，第2式は2次，第3式は3次であるから， 消去法により，\(f\) または \(g\) または \(h\) についての 次数 \(1 \cdot 2 \cdot 3 = 6\) の最終的な方程式が得られることになる． したがって，この方法はもとの4次方程式よりも高次の簡約方程式を導くことになり， 成功する見込みはないように思われる． しかし，その6次の簡約方程式をさらに低い次数に下げることができるということもあり得る． 計算に着手する前に，その可能性をアプリオリに検討しておくのがよいだろう．

42．それには， \[y^{4} + D = 0\] の形に簡約されるために， たとえば \(f\) が根 \(a, b, c, d\) のどのような関数であるべきかを求めなければならない． この \(y\) についての方程式の根は次の4つである（第25項）： \begin{align*} y &= \pm \sqrt[4]{-D}, \\ y &= \pm \sqrt{-1} \, \sqrt[4]{-D}. \end{align*} 簡単のため，\(\sqrt[4]{-D} = k\) とおくと，補助方程式 \[ x^{3} + f x^{2} + g x + h + y = 0 \] において，\(x\) に \(a, b, c, d\) を，\(y\) に \(k, -k, k\sqrt{-1}, -k\sqrt{-1}\) をそれぞれ代入すればよい． すると次の4つの式が得られる： \[ \begin{alignedat}{2} a^{3} +& a^{2} f +& a g +& h + k = 0, \\ b^{3} +& b^{2} f +& b g +& h - k = 0, \\ c^{3} +& c^{2} f +& c g +& h + k \sqrt{-1} = 0, \\ d^{3} +& d^{2} f +& d g +& h - k \sqrt{-1} = 0. \end{alignedat} \] これらから \(f, g, h, k\) の値を決定することができる．

まず，上の2式と下の2式をそれぞれ加えると： \[ \begin{aligned} &a^{3} + b^{3} + (a^{2} + b^{2}) f + (a + b) g + 2h = 0, \\ &c^{3} + d^{3} + (c^{2} + d^{2}) f + (c + d) g + 2h = 0, \end{aligned} \] これらを引き算すれば： \[ a^{3} + b^{3} - c^{3} - d^{3} + (a^{2} + b^{2} - c^{2} - d^{2}) f + (a + b - c - d) g = 0. \] ここで未知数は \(f\) と \(g\) の2つだけとなる．

次に，上の2式と下の2式をそれぞれ引き算すると： \[ \begin{aligned} &a^{3} - b^{3} + (a^{2} - b^{2}) f + (a - b) g + 2k = 0, \\ &c^{3} - d^{3} + (c^{2} - d^{2}) f + (c - d) g + 2k \sqrt{-1} = 0. \end{aligned} \] このうち後者を \(\sqrt{-1}\) 倍して前者に加えると： \[ a^{3} - b^{3} + (c^{3} - d^{3}) \sqrt{-1} + \left[ a^{2} - b^{2} + (c^{2} - d^{2}) \sqrt{-1} \right] f + \left[ a - b + (c - d) \sqrt{-1} \right] g = 0. \] この式と，先に得られた式とを組み合わせれば， \(f\) および \(g\) を決定することができる．

\(g\) を消去すると，\(f\) についての次の式が得られる： \[ f = -\frac{ (a^{3} + b^{3} - c^{3} - d^{3})[a - b + (c - d)\sqrt{-1}] - [a^{3} - b^{3} + (c^{3} - d^{3})\sqrt{-1}](a + b - c - d) }{ (a^{2} + b^{2} - c^{2} - d^{2})[a - b + (c - d)\sqrt{-1}] - [a^{2} - b^{2} + (c^{2} - d^{2})\sqrt{-1}](a + b - c - d) }. \] この式の \(a, b, c, d\) にあらゆる可能な置換を施すことで，\(f\) が取り得るすべての異なる値を直ちに導き出すことができる． 簡略のため，次のように置く： \begin{align*} M\,&=(a^3+b^3 -c^3-d^3)(a-b)-(a^3-b^3)(a+b-c-d),\\ N\,\,&=(a^3+b^3 -c^3-d^3)(c-d)-(c^3-d^3)(a+b-c-d),\\ P\,\,&=(a^2+b^2 -c^2-d^2)(a-b)-(a^2-b^2)(a+b-c-d),\\ Q\,\,&=(a^2+b^2 -c^2-d^2)(c-d)-(c^2-d^2)(a+b-c-d),\\[+5pt] M'&=(a^3+c^3 -b^3-d^3)(a-c)-(a^3-c^3)(a+c-b-d),\\ N'\,&=(a^3+c^3 -b^3-d^3)(b-d)-(b^3-d^3)(a+c-b-d),\\ P'\,&=(a^2+c^2 -b^2-d^2)(a-c)-(a^2-c^2)(a+c-b-d),\\ Q'\,&=(a^2+c^2 -b^2-d^2)(b-d)-(b^2-d^2)(a+c-b-d),\\[+5pt] M''&=(a^3+d^3 -b^3-c^3)(a-d)-(a^3-d^3)(a+d-b-c),\\ N''&=(a^3+d^3 -b^3-c^3)(b\,-c)-(b^3\,-d^3)(a+d-b-c),\,^{\text{[*1]}}\\ P''&=(a^2+d^2 -b^2-c^2)(a-d)-(a^2-d^2)(a+d-b-c),\\ Q''&=(a^2+d^2 -b^2-c^2)(b\,-c)-(b^2\,-c^2)(a+d-b-c). \end{align*} \(f\) の値として次の6つが得られる： \begin{align*} &-\dfrac{M + N \sqrt{-1}}{P + Q \sqrt{-1}}, \qquad -\dfrac{M - N \sqrt{-1}}{P - Q \sqrt{-1}}, \\[+3pt] &-\dfrac{M' + N' \sqrt{-1}}{P' + Q' \sqrt{-1}}, \quad\,\, -\dfrac{M' - N' \sqrt{-1}}{P' - Q' \sqrt{-1}}, \\[+3pt] &-\dfrac{M'' + N'' \sqrt{-1}}{P'' + Q'' \sqrt{-1}}, \quad -\dfrac{M'' - N'' \sqrt{-1}}{P'' - Q'' \sqrt{-1}}. \end{align*} これらが \(f\) についての方程式の根であり，したがってこの方程式は， すでに別の方法で結論づけたとおり，6次方程式であることが確認される．

43．ここで問題となるのは，この6次方程式がより低い次数に簡約できるかどうかである． ところが，先に求めたこの方程式の6つの根の形から判断して， じっさいに簡約が可能であることが示される．これからそのことを証明しよう． たとえば，次の2つの根 \[ -\frac{M + N \sqrt{-1}}{P + Q \sqrt{-1}}, \quad -\frac{M - N \sqrt{-1}}{P - Q \sqrt{-1}} \] が，2次方程式 \[ f^{2} + t f + u = 0 \] の根であると仮定しよう． このとき，方程式の性質から係数 \(t\) および \(u\) は次のように与えられる： \[ t = \frac{M + N \sqrt{-1}}{P + Q \sqrt{-1}} + \frac{M - N \sqrt{-1}}{P - Q \sqrt{-1}}, \quad u = \frac{M + N \sqrt{-1}}{P + Q \sqrt{-1}} \times \frac{M - N \sqrt{-1}}{P - Q \sqrt{-1}}, \] すなわち， \[ t = \frac{2(MP + NQ)}{P^{2} + Q^{2}}, \quad u = \frac{M^{2} + N^{2}}{P^{2} + Q^{2}}. \] さて，私は，以下のことを主張する．この \(t\) および \(u\) は，それぞれ次のような3次方程式の根である： \begin{align*} &\,t^3-\,E\,t^2+F\,t\,-G=0,\\ &u^3-Hu^2+Ku-L=0. \end{align*} ここで \(E, F, G, H, K, L\) は，もとの方程式の係数 \(m, n, p, q\) の有理関数である． このとき，\(t', t'', t'''\) を第1の方程式の3つの根， \(u', u'', u'''\) を第2の方程式の対応する根とすれば， 次の3つの2次方程式が得られる： \begin{align*} &f^2+\,t'\,f\,+\,u'\,=0,\\ &f^2+t''\,f+\,u''=0,\\ &f^2+t'''f+u'''=0. \end{align*} 先ほどの \(f\) についての6次方程式は，これら3つに分解されることになる．

この命題を証明するには，\(t\) および \(u\)，あるいは同じことだが，もとの方程式の根 \(a, b, c, d\) の関数 \[ \frac{MP + NQ}{P^{2} + Q^{2}}, \quad \frac{M^{2} + N^{2}}{P^{2} + Q^{2}} \] が，それらの根の間にあらゆる可能な置換を施したとき，何通りの異なる値をとりうるかを調べればよい． というのも，得られる値は \(t\) および \(u\) についての方程式の根となるからである． まず，\(a, b, c, d\) の置換（順列）の総数は，よく知られているように \(4 \cdot 3 \cdot 2 \cdot 1 = 24\) 通りであるから，一般的には \(t\) および \(u\) についての方程式は24次になるはずである． しかし実際には，これらの置換の中には \(t\) および \(u\) に同じ値を与えるものが複数存在し， それらは除外されるべきである．

たとえば：

1\(^\circ\,\) \(a\) と \(b\) を交換すると，\(N\) および \(Q\) は変わらず，\(M\) および \(P\) は符号が反転するだけなので， \(t\) および \(u\) の値は変わらない．したがって，24通りの置換に対応する24個の値のうち， 12個は他の12個と一致し，\(t\) および \(u\) の有効な値の数はすでに半分に減る．

2\(^\circ\,\) \(c\) と \(d\) を交換すると，\(M\) および \(P\) は変わらず，\(N\) および \(Q\) の符号が反転するだけなので， やはり \(t\) および \(u\) の値は変わらない．この交換は前のものとは独立なので， 同様の理由により，12個の値はさらに6個に減る．

3\(^\circ\,\) \(a\) と \(c\)，\(b\) と \(d\) を同時に交換すると，\(M\) と \(N\) は互いに符号を変えて入れ替わり， \(P\) と \(Q\) も同様に入れ替わるが，\(t\) および \(u\) の値には変化がない． この交換も前のものとは独立なので，6個の値はさらに3個に減る． したがって，\(t\) および \(u\) に対応する24個の値のうち， 実際に異なるのは3つだけであり，それぞれが8回ずつ重複して現れる．

なお，\(a\) と \(d\)，\(b\) と \(c\) を同時に交換する操作も \(t\) および \(u\) の値を変えないが， これはすでに上記の交換に含まれているため，別に数える必要はない．

以上から，\(t\) および \(u\) についての24次方程式は， 実際には3つの異なる根をそれぞれ8重根としてもつことになり， それゆえ，これらの方程式は3次方程式の8乗にほかならない．

44．以上の考察から，\(t\) および \(u\) の異なる値はそれぞれ3つに限られることがアプリオリにわかった． そして，それらの値は次のように与えられる． \(t\) に対しては： \[ \frac{2(MP + NQ)}{P^{2} + Q^{2}}, \quad \frac{2(M'P' + N'Q')}{P'^{2} + Q'^{2}}, \quad \frac{2(M''P'' + N''Q'')}{P''^{2} + Q''^{2}}, \] \(u\) に対しては： \[ \frac{M^{2} + N^{2}}{P^{2} + Q^{2}}, \quad \frac{M'^{2} + N'^{2}}{P'^{2} + Q'^{2}}, \quad \frac{M''^{2} + N''^{2}}{P''^{2} + Q''^{2}}. \] すなわち(第43項)： \[ t' = \frac{2(MP + NQ)}{P^{2} + Q^{2}}, \quad t'' = \frac{2(M'P' + N'Q')}{P'^{2} + Q'^{2}}, \quad t'''= \frac{2(M''P'' + N''Q'')}{P''^{2} + Q''^{2}}, \] そして， \[ u' = \frac{M^{2} + N^{2}}{P^{2} + Q^{2}}, \qquad\quad u'' = \frac{M'^{2} + N'^{2}}{P'^{2} + Q'^{2}}, \qquad\quad\,\, u'''= \frac{M''^{2} + N''^{2}}{P''^{2} + Q''^{2}}.\qquad\quad \] \(t\) および \(u\) についての方程式の係数 \(E, F, \dots\) をこれらの値で表すと，よく知られているように， 以下のようになる： \[ \begin{alignedat}{3} E &= t' + t'' + t''', \qquad &F &= t' t'' + t' t''' + t'' t''', \qquad &G &= t' t'' t''', \\ H &= u' + u'' + u''', \qquad &K &= u' u'' + u' u''' + u'' u''', \qquad &L &= u' u'' u'''. \end{alignedat} \] これらは \(a, b, c, d\) の関数で， \(a, b, c, d\)にどのような置換を施しても不変である． したがって，これらは\(a, b, c, d\)を根に持つ与えられた方程式の係数 \(m, n, p, q\) の有理関数として表すことができる． こうして，ここで問題となっている係数の値を直接求めることができる． この方法はこれまでの考察でもたびたび用いてきたものである．

さらに，\(t\) についての方程式の3つの根 \(t', t'', t'''\) がわかれば， それに対応する \(u\) の根 \(u', u'', u'''\) は，方程式を解くことなく求めることができる． 実際，次の3つの式を考える： \[ \begin{aligned} &u' + u'' + u''', \\ &t' u' + t'' u'' + t''' u''', \\ &t'^{2} u' + t''^{2} u'' + t'''^{2} u'''. \end{aligned} \] そして，\(t', t'', t'''\) および \(u', u'', u'''\) に上記の \(a, b, c, d\) による式を代入すれば， そこから得られる \(a, b, c, d\) についての関数は，\(a, b, c, d\) にどのような置換を施しても形を変えないことが容易にわかる． したがって，これらの値は常に与えられた方程式の係数 \(m, n, p, q\) の有理関数として表すことができる． これらの式の値がわかれば，得られた3つの方程式から未知数 \(u', u'', u'''\) は容易に決定できる（第IV節を参照）．

45．以上により，この方法によって導かれる6次の簡約方程式は， 3次へと簡約できることがアプリオリに確かめられたので， ここではこの方法が要求する計算手順を見ていくことにしよう． まず，第40項で導入した補助方程式 \[ x^{3} + f x^{2} + g x + h + y = 0 \] を取り上げ，この方程式が，もとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] と共通の根をもつための必要条件を，通常の方法（第11項）によって求める． まず，簡単のため \(h' = h + y\) とおき， 多項式 \[x^{4} + m x^{3} + n x^{2} + p x + q\] を多項式 \[x^{3} + f x^{2} + g x + h'\] で割る． 商は無視して，余りは次の形になる： \[ M x^{2} + N' x + P', \] ただし， \[ \begin{aligned} M &= n - g - f(m - f), \\ N' &= p - h' - g(m - f), \\ P' &= q - h'(m - f). \end{aligned} \] 次に，\(x^{3} + f x^{2} + g x + h'\) を \(M x^{2} + N' x + P'\) で割ると，次の余りを得る： \[ \left( g - \frac{P' + N' f}{M} + \frac{N'^{2}}{M^{2}} \right) x + h' - \frac{P' f}{M} + \frac{N' P'}{M^{2}} = 0. \] これは \(x\) の1次式となるので，求める共通因子でなければならない．この式を0とすることで， \[ x = -\frac{M^{2} h' - M P' f + N' P'}{M^{2} g - M(P' + N' f) + N'^{2}}. \] この値を，先ほどの方程式 \[ M x^{2} + N' x + P' = 0 \] に代入すれば，求める条件が得られる．

ここで次のように置く： \[ \begin{aligned} N &= p - h - g(m - f), \\ P &= q - h(m - f), \end{aligned} \] また，\(h' = h + y\) であるから， \[ \begin{aligned} N' &= N - y, \\ P' &= P - (m - f)y. \end{aligned} \] これらを \(x\) の式に代入し，さらに次のように置く： \[ \begin{aligned} Q &= M^{2} h - M P f + N P, \\ R &= M^{2} + (M f - N) f - P,\,\,^{\text{[*1]}}\\ S &= M^{2} g - M(P + N f) + N^{2}, \\ T &= M m - 2 N, \end{aligned}\] すると，次のようになる： \[ x = -\frac{Q + R y + (m - f) y^{2}}{S + T y + y^{2}}. \] 条件式は次のようになる： \[ \begin{aligned} M \left[ Q + R y + (m - f) y^{2} \right]^{2} + (y - N) \left[ Q + R y + (m - f) y^{2} \right] (S + T y + y^{2}) \qquad&\\ + \left[ P - (m - f) y \right] (S + T y + y^{2}) = 0.&\,\,^{\text{[*2]}} \end{aligned} \] この式を展開し，\(y\) の冪に従って整理すれば，すでに上で示したように，次の形になる： \[ y^{4} + A y^{3} + B y^{2} + C y + D = 0. \]

この \(y\) についての方程式において，係数 \(A, B, C, D\) は \(f, g, h\) の3つの未定係数に関する多項式であり， すでにアプリオリに証明したように， それぞれの次数は \(A\) において1次，\(B\) において2次，…等々となる． したがって，この方程式を2項式に簡約するには， \[ A = 0, \quad B = 0, \quad C = 0 \] とすればよい． この3つの方程式から，まず \(g\) および \(h\) を \(f\) の式として求め， そこから，最終的に \(f\) についての6次方程式が得られる． ただし，これはすでに示したように，3次に簡約可能である． 実際，この6次方程式を \[ f^{2} + t f + u = 0 \] という2次方程式で割ると，割り切れるための条件として \(t\) と \(u\) の間に2つの関係式が得られる． これにより，まず \(u\) を \(t\) の式で表す[*3]． そして \(t\) について得られる最終的な方程式は 3次にすぎない． この3次方程式を解けば \(t\) がわかり，そこから \(u\) が決まる． さらに上の2次方程式を解くことで \(f\) が得られる． そして \(g\) および \(h\) は1次の関係式[*4]から求まる． このようにして，\(D, Q, R, S, T\) のすべての値が求まる．

さて，\(y\) についての方程式は中間項を消すことで， \[y^{4} + D = 0\] に簡約されたので， \(y\)の値は次の4つとなる： \[ \pm \sqrt[4]{-D}, \quad \pm \sqrt[4]{-D} \, \sqrt{-1}. \] これらを順に上の \(x\) の式に代入すれば， もとの方程式の4つの根が得られる． もっとも，この計算はかなり複雑な式に至るため， ここでは以上の指摘にとどめ， むしろこれを簡単にする方法を探ることにする．

46．根 \(x\) が次の形 \[ x = -\frac{Q + R y + (m - f) y^{2}}{S + T y + y^{2}} \] で与えられ，\(y\) は2項式 \[y^{4} + D = 0\] によって定まるので， \(x\) の式は次のようなより単純な形 \[ x = a + b y + c y^{2} + d y^{3} \] に簡約できることが容易にわかる．ここで \(a, b, c, d\) は \(Q, R, \dots\) に依存する係数である． 実際，分数 \[ \frac{Q + R y + (m - f) y^{2}}{S + T y + y^{2}} \] の分子と分母に \(S - T y + y^{2}\) を掛けると，分母は \[ (S + y^{2})^{2} - T^{2} y^{2},\quad\text{すなわち}\quad S^{2} + (2S - T^{2}) y^{2} + y^{4} \] となる．ここで \(y^{4}\) に \(-D\) を代入すれば， \[ S^{2} - D + (2S - T^{2}) y^{2} \] となる． さらに，この式で分子と分母に\(S^{2} - D - (2S - T^{2}) y^{2}\) を掛けると，分母は \[ (S^{2} - D)^{2} - (2S - T^{2})^{2} y^{4} \] となり，\(y^{4} = -D\) を代入すれば， \[ (S^{2} - D)^{2} + D (2S - T^{2})^{2} \] となって，\(y\) が一掃される． したがって，\(x\) の式の分母から \(y\) を消すには，分子と分母に \[ (S - T y + y^{2}) \left[ S^{2} - D - (2S - T^{2}) y^{2} \right] \] を掛ければよい． この操作によって分子は \(y\) の6次式となるが， \(y^{4}\)に\(-D\), \(y^{5}\)に\(-D y\), \(y^{6}\)に\(-D y^{2}\) を代入すれば， \(y, y^{2}, y^{3}\) のみを含む式になる． したがって，\(x\) は \[ x = a + b y + c y^{2} + d y^{3} \] の形で表されることになる．

いま，\(y\)の値として\(y^{4} + D = 0\) の根を代入すれば，\(x\) の4つの値が得られ， それらはもとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] の根であるから，この方程式は次の2つの式 \[ x = a + b y + c y^{2} + d y^{3}, \quad y^{4} + D = 0 \] から \(y\) を消去して得られるものと見なすことができる． したがって，対応する項を比較すれば，4つの方程式が得られ，それによって，5つの係数 \(a, b, c, d, D\) のうち任意の1つを自由に定め，残る4つを決定することができる．

これは，4次方程式の解法に関して，前掲の論文（第18項）においてオイラー氏およびベズー氏が提案した方法である．

オイラー氏は \(c = 1\) とおき，残りの3つの未定係数 \(a, b, d\) を消去することによって，\(D\) についての3次の簡約方程式を得た． これに対してベズー氏は，まず \(D = -1\) とおき，\(c\) についての6次の簡約方程式を得たが，この方程式は未知数の奇数次の冪を含まないため，3次方程式と同様の方法で解くことができる． さらにベズー氏は，\(c\) の代わりに \(b\) または \(d\) を求めようとすれば，指数が4の倍数であるような24次の簡約方程式に至ることを示しており，したがって6次方程式として解くことになる． 加えて彼は，\(b d\) を根とする簡約方程式を求めれば，それは3次方程式にすぎないことを示している． このことによって，彼は，\(b\) または \(d\) についての簡約方程式も，\(b d\) についての方程式を用いれば，指数が4の倍数であるような3つの8次方程式（これらは，2次方程式と同様の方法で解くことができる）に分解できて，結局は3次方程式の困難さに帰着することを示している．

これらの結果については，引用された論文を参照できない場合でも読者自身で容易に導出できるであろうから，ここではその指摘にとどめ，これまでと同様に，なぜこのような結果が生じるのかという根拠をアプリオリに探求することにしよう．

47．\(x', x'', x''', x^{\mathrm{IV}}\) を \(x\) の4つの値，すなわち与えられた方程式の根とする． \(y^{4} + D = 0\)から得られる \(y\)の4つの値は \(\pm \sqrt[4]{-D}\) および \(\pm \sqrt[4]{-D} \, \sqrt{-1}\) なので， これらを式 \[x = a + b y + c y^{2} + d y^{3}\] に順に代入することによって，次の4つの式が得られる： \begin{align*} x'\,\,&= a + b\sqrt[4]{-D} + c\sqrt[4]{D^2} + d\sqrt[4]{-D^3},\\ x''\,&= a - b\sqrt[4]{-D} + c\sqrt[4]{D^2} - d\sqrt[4]{-D^3},\\ x'''&= a + b\sqrt[4]{-D}\,\sqrt{-1} - c\sqrt[4]{D^2} - d\sqrt[4]{-D^3}\,\sqrt{-1},\\ x^{\mathrm{IV}} &= a - b\sqrt[4]{-D}\,\sqrt{-1} - c\sqrt[4]{D^2} + d\sqrt[4]{-D^3}\,\sqrt{-1}. \end{align*} まず，これら4つの式をすべて加えると： \[ x' + x'' + x''' + x^{\mathrm{IV}} = 4a = -m, \] したがって， \[ a = \frac{-m}{4}. \] 次に，最初の2つと後の2つをそれぞれ加えると： \begin{align*} x'\,+\,x''\,&= 2a+2 c\sqrt[4]{D^2},\\ x'''+x^{\mathrm{IV}}&= 2a - 2c\sqrt[4]{D^2}, \end{align*} よって， \[ c \sqrt[4]{D^{2}} = c \sqrt{-D} = \frac{x' + x'' - x''' - x^{\mathrm{IV}}}{4}. \] したがって，オイラー氏のように \(c = 1\) とすれば， \[ -D = \frac{(x' + x'' - x''' - x^{\mathrm{IV}})^2}{16}. \] そして，この \(D\) の式から，オイラー氏が見出したように，\(D\) についての方程式が実際に3次であることが容易にわかる． なぜなら，それは，以前（第32項）に導出された \(t\) についての簡約方程式に他ならず， そこの \(t\) を \(-16 D\) に置き換えたものだからである． というのも，そこでは \[ t = s^{2} = (a + b - c - d)^{2} \] とされており，\(a, b, c, d\) は， 現在我々が \(x', x'', x''', x^{\mathrm{IV}}\) と記している量， すなわち与えられた方程式の4つの根を表していたのである．

しかし，もしオイラー氏が \(c = 1\) ではなく \(b = 1\) と仮定していたならば， \(D\) についての簡約方程式はもはや3次ではなく，6次に達していたであろう．

実際，上記の4式から最初の2つの差と後の2つの差をとると： \begin{align*} x'\,-\,x''\,\,&=\,\,2b\sqrt[4]{-D} +2d\sqrt[4]{-D^3},\\ x'''-x^{\mathrm{IV}}&= \left[2b\sqrt[4]{-D} -2d\sqrt[4]{-D^3}\right]\sqrt{-1}, \end{align*} したがって， \[ \begin{aligned} b \sqrt[4]{-D} &= \frac{x' - x'' - (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4}, \\[+3pt] d \sqrt[4]{-D^{3}} &= \frac{x' - x'' + (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4}. \end{aligned} \] よって，\(b = 1\) とすれば，4乗して， \[ -D = \left[ \frac{x' - x'' - (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4} \right]^{4}. \] 次項で示されるように，この量は6次の方程式に従う．

48．ベズー氏のように \(D = -1\) とおけば，先の式により， \[ c = \frac{x' - x'' + x''' - x^{\mathrm{IV}}}{4}. \] ここからまず，この著者が見出したとおり，\(c\) についての簡約方程式は6次であり，すべての項の次数が偶数であることが結論される． というのも，オイラー氏の仮定における \(-D\) の値は，現在の仮定における \(c^{2}\) の値と一致するため， オイラー氏の簡約方程式において \(D\) の代わりに \(-c^{2}\) を代入すれば， ベズー氏の \(c\) についての簡約方程式が得られることになる． したがって，この方程式は6次であり，3次方程式と同様の方法で解くことができる． なお，この \(c\) についての簡約方程式は，第29項の \(z\) についての簡約方程式と同じものであり， そこでの \(z\) の代わりに \(-2c\) を代入すればよい．

さて，ベズー氏と同様に \(D = -1\) としたまま，\(b\) および \(d\) についての簡約方程式の形を調べてみよう． この仮定のもとで，第47項の式により： \[ \begin{aligned} b &= \frac{x' - x'' - (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4}, \\[+3pt] d &= \frac{x' - x'' + (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4}. \end{aligned} \] ここから，\(x', x'', x''', x^{\mathrm{IV}}\) の4つの根にあらゆる可能な置換を施すことにより， \(b\) および \(d\) のすべての値が得られる． そして，それらの値の個数と形状から，\(b\) および \(d\) を決定する方程式の次数と性質を判断することができる． すなわち：

1\(^\circ\,\) \(b\) についての方程式は \(d\) についての方程式と同じである． なぜなら，\(d\) の値は \(b\) の値において \(x'''\) と \(x^{\mathrm{IV}}\) を入れ替えることで得られるため， \(b\) および \(d\) の値は同一の方程式の根となるからである．

2\(^\circ\,\) この方程式は一般に\(4 \cdot 3 \cdot 2 \cdot 1\)，すなわち，24次である． というのも，\(x', x'', x''', x^{\mathrm{IV}}\) の4つの量に対して可能な置換（順列）の数がそれだけあるからである．

3\(^\circ\,\) この24次方程式は，すべての項の次数が4の倍数である． なぜなら，\(b\) がその根の1つであるとすれば，\(-b\), \(b \sqrt{-1}\), \(-b \sqrt{-1}\) もまたその根となるからである． 実際，上と同様に \[ b = \frac{x' - x'' - (x''' - x^{\mathrm{IV}}) \sqrt{-1}}{4} \] とすれば，\(x'\) と \(x''\)，\(x'''\) と \(x^{\mathrm{IV}}\) を入れ替えることで \(b\) は \(-b\) となり， \(x'\) と \(x'''\)，\(x''\) と \(x^{\mathrm{IV}}\) を入れ替えることで \(b\) は \(b \sqrt{-1}\) となり， \(x'\) と \(x^{\mathrm{IV}}\)，\(x''\) と \(x'''\) を入れ替えることで \(b\) は \(-b \sqrt{-1}\) となることが明らかである． したがって，\(b\) を負にしたり，\(b\) の代わりに \(\pm b \sqrt{-1}\) を代入しても， 方程式の形が変わらない必要がある． このことは，その方程式が \(b\) の奇数次や，2の奇数倍の次数の項を含まないことを意味する． よって，\(b^{4} = v\) とおけば，\(v\) についての6次の簡約方程式が得られる． そして，この \(v\) についての簡約方程式は， 第47項における仮定 \(b = 1\) のもとでの \(-D\) についての簡約方程式と同じものである． というのも，\(-D\) の値は上記の \(b^{4}\) の値と一致するからである．

ここで，第42項で用いたのと同様の方法によって，この \(v\) についての方程式が 3次の簡約方程式を用いて3つの2次方程式に分解できることを示すことも可能だが， それよりも簡単な次の方法で証明することができる．

\(b\) および \(d\) の積をとると： \[ bd = \frac{(x' - x'')^{2} + (x''' - x^{\mathrm{IV}})^{2}}{16}, \] ここで， \begin{align*} (x' - x'')^{2} + (x''' - x^{\mathrm{IV}})^{2} &= x'^{2} + x''^{2} + x'''^{2} + x^{\mathrm{IV}2} - 2(x' x'' + x''' x^{\mathrm{IV}})\\ &= m^{2} - 2n - 2(x' x'' + x''' x^{\mathrm{IV}}). \end{align*} \(x' x'' + x''' x^{\mathrm{IV}}\) は第30項における量 \(u\) に一致し， \(u\) は3次方程式に従うことがすでに示されている． したがって，\(bd\) についての方程式もまた3次である． 実際， \[ bd = \frac{m^{2} - 2n - 2u}{16} \] であるから，\(u\) についての方程式において \(u\) の代わりに \(\frac{m^{2} - 2n - 16bd}{2}\) を代入すれば，\(bd\) についての方程式が得られる．

いま，この \(bd\) についての方程式の根を \(\rho', \rho'', \rho'''\) とすれば， \[ \begin{aligned} \rho' &= \frac{m^{2} - 2n}{16} - \frac{x' x'' + x''' x^{\mathrm{IV}}}{8} = bd, \\ \rho'' &= \frac{m^{2} - 2n}{16} - \frac{x' x''' + x'' x^{\mathrm{IV}}}{8}, \\ \rho''' &= \frac{m^{2} - 2n}{16} - \frac{x' x^{\mathrm{IV}} + x'' x'''}{8} \end{aligned} \] を得る（第30項）．また，第48項から得られる次の2つの式 \[ x' - x'' = 2(b + d), \quad x''' - x^{\mathrm{IV}} = 2(b - d) \sqrt{-1} \] を掛け合わせると， \[ x' x''' + x'' x^{\mathrm{IV}} - x'' x''' - x' x^{\mathrm{IV}} = 4(b^{2} - d^{2}) \sqrt{-1}, \] したがって， \[ 4(b^{2} - d^{2}) \sqrt{-1} = 8(\rho''' - \rho''). \] 両辺を2乗すれば， \[ b^{4} - 2b^{2} d^{2} + d^{4} = -4(\rho''' - \rho'')^{2}. \] また，\(bd = \rho'\) であるから， \[ b^{4} + d^{4} = 2\rho'^{2} - 4(\rho''' - \rho'')^{2}, \] さらに，\(d = \frac{\rho'}{b}\) なので， \[ b^{4} + \frac{\rho'^{4}}{b^{4}} = 2\rho'^{2} - 4(\rho''' - \rho'')^{2}, \] よって， \[ b^{8} - 2\left[ \rho'^{2} - 2(\rho''' - \rho'')^{2} \right] b^{4} + \rho'^{4} = 0. \] これは8次方程式であり，2次方程式と同様の方法で解くことができる． この結果はベズー氏の得たものと一致する．

なお，\(b\) についての簡約方程式に関連して次のことを付言しておくのが適切だろう． すなわち，\(x^{4} - 1 = 0\) の4つの根を \(1, \alpha, \alpha^{2}, \alpha^{3}\) とすれば（第25項）， \[ b = \frac{x' + \alpha x^{\mathrm{IV}} + \alpha^{2} x'' + \alpha^{3} x'''}{4}, \] あるいは，\(x^{\mathrm{IV}}\) を \(x''\) に，\(x''\) を \(x'''\) に，\(x'''\) を \(x^{\mathrm{IV}}\) に入れ替えれば， \[ b = \frac{x' + \alpha x'' + \alpha^{2} x''' + \alpha^{3} x^{\mathrm{IV}}}{4}. \] この式は，3次方程式を簡約するための式（第6項および第19項）と類似しており， この最後の方法によって導かれる4次方程式の解法と， 3次方程式の解法との間にある類似性を示すものである．

49．第46項の式 \begin{align*} &x = a + b y + c y^{2} + d y^{3},\\ &y^{4} + D = 0 \end{align*} を取り上げ，\(y^{2} = z\) とおけば，次の2つの式が得られる： \begin{align*} &x = (a + c z) + (b + d z) \sqrt{z}, \\ &z^{2} + D = 0. \end{align*} 前者の式で根号を解消すれば， \[ (x - a - c z)^{2} - (b + d z)^{2}z = 0, \] \(z^{2} = -D\) であることから，次の形に簡約される： \[ x^{2} + (f + g z) x + h + k z = 0. \] このようにして，次の2式が得られる： \begin{align*} &z^{2} + D = 0, \\ &x^{2} + (f + g z) x + h + k z = 0. \end{align*} これらから \(z\) を消去すれば，もとの方程式 \[ x^{4} + m x^{3} + n x^{2} + p x + q = 0 \] と比較可能な4次方程式が得られる． したがって，対応する項を比較することにより，\(f, g, h, k, D\) のうち任意の1つを自由に定め，残る4つの係数は決定できる．

この方法は，ベズー氏が1762年の『方程式について（sur les équations）』の末尾で示した方法と同じものであり， また彼が1765年の論文の548ページでも再掲しており， これは次数が合成数のあらゆる方程式に拡張可能な一般的方法の一例として提示されている． これらのうち前者では，著者はまず \(g = -1\) と仮定し，\(k\) についての3次の最終方程式を得ている． 後者では \(D = -1\) と仮定し，\(g\) についての6次の最終方程式を得ているが， その指数はすべて偶数であるため，3次方程式と同様の方法で解くことができる．

これらの結果の理由を理解するには，\(z = \pm \sqrt{-D}\) であることに注意すれば十分である． このとき，次の2つの方程式が得られる： \[ \begin{aligned} x^{2} + (f + g \sqrt{-D}) x + h + k \sqrt{-D} &= 0, \\ x^{2} + (f - g \sqrt{-D}) x + h - k \sqrt{-D} &= 0. \end{aligned} \] これらの積は与えられた4次方程式を与えるはずである． したがって，これらのうち一方は与えられた方程式の2つの根を含み， 他方は残りの2つの根を含むことになる． よって，方程式の性質から次の関係が得られる： \[ \begin{alignedat}{2} -f - g \sqrt{-D} &= x' + x'', \qquad & h + k \sqrt{-D} &= x' x'', \\ -f + g \sqrt{-D} &= x''' + x^{\mathrm{IV}}, & h - k \sqrt{-D} &= x''' x^{\mathrm{IV}}, \end{alignedat} \] したがって， \[ \begin{aligned} -2g \sqrt{-D} &= x' + x'' - x''' - x^{\mathrm{IV}}, \\ 2k \sqrt{-D} &= x' x'' - x''' x^{\mathrm{IV}}. \end{aligned} \] ここでまず \(g = -1\) とおき，最初の式から \(\sqrt{-D}\) の値を求めてそれを2番目の式に代入すれば， \[ k = \frac{x' x'' - x''' x^{\mathrm{IV}}}{x' + x'' - x''' - x^{\mathrm{IV}}}. \] このことから，\(k\) についての方程式は3次であることがわかる． というのも，\(x', x'', x''', x^{\mathrm{IV}}\) の4つの根にどのような置換を施しても， \(k\) の値は次の3通りにしかならないからである： \[ \begin{aligned} &\frac{x' x'' - x''' x^{\mathrm{IV}}}{x' + x'' - x''' - x^{\mathrm{IV}}}, \\[+3pt] &\frac{x' x''' - x'' x^{\mathrm{IV}}}{x' + x''' - x'' - x^{\mathrm{IV}}}, \\[+3pt] &\frac{x' x^{\mathrm{IV}} - x'' x'''}{x' + x^{\mathrm{IV}} - x'' - x'''}. \end{aligned} \] もし望むなら，これらの値から，\(k\) についての方程式を直接導くことも可能である．

次に \(D = -1\) とおけば， \[ g = \frac{x^{\mathrm{IV}} + x''' - x' - x''}{2} \] となり，この \(g\) の値は第29項の量 \(z\) に一致するため， 第32項で得られた結果を適用することができる． この仮定 \(D = -1\) のもとで，\(g\) の代わりに \(k\) を求めるとすれば， \[ k = \frac{x' x'' - x''' x^{\mathrm{IV}}}{2} \] となり，\(k\) についての方程式もまた6次で，すべての項の次数は偶数である． その根は次のようになる： \begin{align*} \frac{x' x'' - x''' x^{\mathrm{IV}}}{2},\quad \frac{x' x''' - x'' x^{\mathrm{IV}}}{2},\quad \frac{x' x^{\mathrm{IV}} - x'' x'''}{2}, \\[+3pt] \frac{x'' x''' - x' x^{\mathrm{IV}}}{2},\quad \frac{x'' x^{\mathrm{IV}} - x' x'''}{2},\quad \frac{x''' x^{\mathrm{IV}} - x' x''}{2}. \end{align*} なお，この \(k\) についての方程式は，第30項の \(u\) についての方程式から容易に導出できる． というのも， \[ k = \frac{x' x'' - x''' x^{\mathrm{IV}}}{2}, \quad u = x' x'' + x''' x^{\mathrm{IV}} \] であり（ここでの \(x', x'', x''', x^{\mathrm{IV}}\) は第30項における \(a, b, c, d\) に対応し， 与えられた方程式の根を表す），次の関係が成り立つ： \[ u^{2} - 4k^{2} = 4 x' x'' x''' x^{\mathrm{IV}} = 4q, \] したがって， \[ u = 2 \sqrt{q + k^{2}}. \] この値を \(u\) についての方程式に代入し，根号を解消すれば， \(k\) についての6次方程式が得られ，そのすべての項の次数は2の倍数となる．

50．ここで，4次方程式の解法に関する諸方法の分析を終えることにしよう． 我々はこれらの方法を互いに比較し，それらの関連性と相互依存性を示しただけでなく，何よりも重要な点として， ある方法が3次の簡約方程式に， また別の方法が6次の簡約方程式（しかし，それは3次へと低次化され得る）に導く理由をアプリオリに与えた． そして読者は，このことが一般的に，それら簡約方程式の根が \(x', x'', x''', x^{\mathrm{IV}}\) の関数――それも，この4つの量にあらゆる可能な置換を施したとき， たとえば \(x'x'' + x'''x^{\mathrm{IV}}\)のように3つの異なる値しか生じないとか， あるいは \(x' + x'' - x''' - x^{\mathrm{IV}}\)のように6つの値が生じるがそれらは符号が逆の対になるとか， または第42項で得られた関数のように6つの値が生じるが，それらを3つの対に分けて，各対の和または積を取ったとき， どのような置換を施してもこれら3つの和または積が常に不変[*1]となる〔とかの性質をもつ〕関数である事実から生じることに気づいたはずである． このような関数の存在こそが，4次方程式の一般的な解法が可能となる唯一の根拠に他ならないのである(*)．
