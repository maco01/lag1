方程式の代数的解法に関する省察(*)
ジョゼフ＝ルイ・ラグランジュ
［ベルリン王立科学文芸アカデミー新紀要，1770年および1771年(**)］

方程式論は解析のすべての分野の中で，その重要性と，最初の創始者たちがそこにおいて成し遂げた進歩の迅速さとによって，最も完成度を高められるはずの分野であると考えられていた．しかしながら，その後に現れた幾何学者たちがこの主題に取り組むことをやめなかったとはいえ，彼らの努力が望まれたような成果を収めたとは到底言いがたい． 実際，方程式の性質，その変形，2つまたはそれ以上の根が等しくなるために必要な条件，あるいはそれらの根が互いにある関係を持つための条件，およびそれらの根を見出す方法，虚根の形，そして実数でありながら虚の形で現れる根の値を求める方法，等々に関しては，ほとんどすべてが尽くされたと言ってよい．また，ある方程式のすべての根が実数であるか否かを判別するための一般的な規則，および前者の場合において正の根と負の根がそれぞれいくつ存在するかを知るための規則も発見されている． しかしながら，今日に至るまで，虚根を含むはずの方程式において，その虚根の個数を知るための一般的な規則は得られていない．たとえ実根と虚根の個数が既知であったとしても，実根のうち正のものと負のものがそれぞれいくつあるかを知るための規則も得られていない．さらには，方程式が奇数次であるか，あるいはその定数項が負である場合を除いては，提示された任意の方程式が果たして実数根を持つか否かを確認するための規則さえ，我々は持っていないのである．

もちろん，方程式の係数の数値的な値が既知である場合には，虚根および正の実根や負の実根の個数を常に求めることは可能である．この目的のため，また各根の値に望むだけ近づくために，私が他のところで与えた方法は，私の見るところ，何ら不満の余地を残すものではない．しかし，ここでの問題は文字係数の方程式に関するものであり，ある次数の方程式において，その根の性質に応じて，係数どうしのあいだに成立すべき条件を見出すことである．

文字係数の方程式の解法に関して言えば，われわれはカルダノの時代――彼が最初に3次方程式および4次方程式の解法を公にした時代――から，ほとんど前進していない．この分野におけるイタリアの解析学者たちの最初の成功は，この主題においてなしうる発見の限界を示していたように思われる．少なくとも確かなのは，代数学のこの分野の限界を押し広げようとしてこれまでになされたすべての試みは，3次および4次の方程式に対する新たな解法を見出すことにしか役立っておらず，それらのいずれもが，一般にはより高次の方程式に適用できるものではなかったことが確かだからである．

本稿において私が意図するのは，これまでに見出された方程式の代数的解法の諸方法を検討し，それらを一般的な原理に還元し，そしてアプリオリに，それらの方法が3次 および4次の方程式に対しては成功し，それ以上の次数に対しては通用しなくなる理由を明らかにすることである．

この検討は2重の利点をもつだろう．一方では，3次および4次の方程式に対して知られている解法に，より大きな明晰さをもたらすことに資するであろうし，他方では，高次の方程式の解法に取り組もうとする者にとって有益であり，この目的のためのさまざまな観点を提供し，とりわけ，多くの無益な試行錯誤や手順を省く助けとなるであろう．

第I節
3次方程式の解法について．

1．2次方程式の解法はきわめて容易であり，その特徴も極度の単純さにあるにすぎないので，私はまず3次方程式から本題に入ることにする．3次方程式の解法には，自然には現れない特別な工夫が必要とされるからである．

3次方程式の一般形を次のようにおこう： \[ x^{3} + mx^{2} + nx + p = 0. \] そして，任意の方程式において第2項を消すには，その根に第2項の係数を最高次の次数で割ったものを加えればよいことが知られているので，簡単のため，まず \(m = 0\) と仮定してよい．すると，この方程式は次の形に帰着される： \[ x^{3} + nx + p = 0. \] 3次方程式はこの形において，最初にシピオーネ・デル・フェッロおよびタルタリアによって扱われ，彼らによってその解法が与えられたが，彼らがどのような道筋でそこに到達したのかは知られていない． この解法に至る最も自然な方法は，私にはフッデが考案したもののように思われる．それは，根を2つの未知量の和とおくことで，方程式を2つの部分に分け，それらの未知量が，2次方程式として解くことのできる1つの方程式にのみ依存するようにする，というものである．

この方法に従えば，\(x = y + z\) とおくことになる．これをもとの方程式に代入すれば，次の形に帰着される： \[ y^{3} + 3y^{2}z + 3yz^{2} + z^{3} + n(y + z) + p = 0. \] この式は，次のようなより簡潔な形に書き換えられる： \[ y^{3} + z^{3} + p + (y + z)(3yz + n) = 0. \] ここで，次の2つの方程式を別々に考えることにしよう： \begin{gather*} y^{3} + z^{3} + p = 0,\\ 3yz + n = 0. \end{gather*} 後者から \[z = -\frac{n}{3y}\] を得る．これを前者に代入すれば， \[ y^{3} - \frac{n^{3}}{27y^{3}} + p = 0, \] すなわち， \[ y^{6} + py^{3} - \frac{n^{3}}{27} = 0. \] この方程式は確かに6次方程式ではあるが，未知数の2つの冪しか含まず，その一方の指数は他方のちょうど2倍であるから，これは2次方程式と同様に解くことができるのは明らかである． 実際，まず， \[ y^{3} = -\frac{p}{2} \pm \sqrt{\frac{p^{2}}{4} + \frac{n^{3}}{27}} \] となるから， \[ y = \sqrt[3]{ -\frac{p}{2} \pm \sqrt{ \frac{p^{2}}{4} + \frac{n^{3}}{27} } }. \] このようにして \(y\) および \(z\) が求められ，そこから \[ x = y + z = y - \frac{n}{3y} \] が得られる．

2．この解法については，いくつかの注目すべき点がある．まず明らかなのは，\(y\) は6次方程式に従うので，6つの値をもつはずであるということである．したがって，\(x\) もまた6つの値をもつことになる．しかし，\(x\) は3次方程式の根であるから，その異なる値は3つしかありえない．ゆえに，この6つの値は3つに集約され，各根が2重根として現れなければならない． このことは，次の2つの方程式から \(y\) を消去することで，計算によっても確かめられる： \[ y^{6} + py^{3} - \frac{n^{3}}{27} = 0,\quad x = y - \frac{n}{3y}. \] より一般的な形として， \[x = y - \frac{k}{y},\quad\text{すなわち}\quad y^{2} - xy - k = 0\] とおこう．すると \[y^{2} = xy + k,\] したがって， \begin{align*} y^{3} &= xy^{2} + ky = y(x^{2} + k) + kx,\\[+3pt] y^{6} &= y^{2}(x^{2} + k)^{2} + 2kx(x^{2} + k)y + k^{2}x^{2} \\ &= yx(x^{2} + k)(x^{2} + 3k) + kx^{4} + 3k^{2}x^{2} + k^{3}. \end{align*} これらの \(y^{3}\) および \(y^{6}\) の値を代入すれば，次の式が得られる： \[ y(x^{2} + k)(x^{3} + 3kx + p) + kx(x^{3} + 3kx + p) + k^{3} - \frac{n^{3}}{27} = 0. \] 簡略化のため， \[ \frac{n^{3}}{27} - k^{3} = h,\quad x^{3} + 3kx + p = X \] とおけば， \[ [y(x^{2} + k) + kx]X - h = 0, \] すなわち， \[ y = \frac{\dfrac{h}{X} - kx}{x^{2} + k}. \] この \(y\) の値を \[y^{2} - xy - k = 0\] に代入すれば， \[ \left( \frac{h}{X} - kx \right)^{2} - x\left( \frac{h}{X} - kx \right)(x^{2} + k) - k(x^{2} + k)^{2} = 0, \] これは次の形に整理される： \[ k^{3}X^{2} + hXx(x^{2} + 3k) - h^{2} = 0. \] また，\(x(x^{2} + 3k) = X - p\) であることから， \[ \frac{n^{3}}{27}X^{2} - hpX - h^{2} = 0, \] すなわち， \[ \left( X - \frac{27hp}{2n^{3}} \right)^{2} - \frac{27h^{2}}{n^{3}}\left( 1 + \frac{27p^{2}}{4n^{3}} \right) = 0. \] ここで，\(x = y - \frac{n}{3y}\) を得るために \(k = \frac{n}{3}\) とすれば，\(h = 0\) となり，上の方程式は \[X^{2} = 0,\] すなわち， \[ (x^{3} + nx + p)^{2} = 0 \] となる．この方程式は，もとの3次方程式と同じ根をもつが，それぞれの根は2重根として現れることになる． したがって，3次方程式の解法は，厳密に言えば6次方程式の解法であり，これは2次方程式のようにその次数自体で完結した解法がある場合とは異なる．そしてこの〔次数の上昇という〕不都合は，高次の方程式においてはさらに深刻なものとなることが，後に明らかになるだろう．

3．さて，\(y\) の6つの値のうち，\(x\) の異なる値を与えるのは3つだけであるから，これらの値を区別する必要がある．そのためには，\(y\) の6つの値それぞれの具体的な表現を求めなければならない． ここで，\(1, \alpha, \beta\) を \(x^{3} - 1 = 0\) の3つの根，すなわち \(1\) の3つの立方根とすれば，次のように置くことで，\(y\) の6つの値は簡潔に表される．略記のため， \(\frac{p^{2}}{4} + \frac{n^{3}}{27}=q\) とおけば，\(y\) の6つの値は次のようになる： \[ \sqrt[3]{ -\frac{p}{2} \pm \sqrt{q} },\quad \alpha \sqrt[3]{ -\frac{p}{2} \pm \sqrt{q} },\quad \beta \sqrt[3]{ -\frac{p}{2} \pm \sqrt{q} }. \] このとき，これに対応する \(z = -\frac{n}{3y}\) の値は，関係式 \[ \sqrt[3]{ -\frac{p}{2} \pm \sqrt{q} } \,\, \sqrt[3]{ -\frac{p}{2} \mp \sqrt{q} } = \sqrt[3]{ \frac{p^{2}}{4} - q } = -\frac{n}{3}, \] すなわち， \[ \sqrt[3]{ -\frac{p}{2} \mp \sqrt{q} } = -\frac{n}{3 \sqrt[3]{ -\dfrac{p}{2} \pm \sqrt{q} }} \] により導かれ，次のように表される： \[ \sqrt[3]{ -\frac{p}{2} \mp \sqrt{q} },\quad \frac{1}{\alpha} \sqrt[3]{ -\frac{p}{2} \mp \sqrt{q} },\quad \frac{1}{\beta} \sqrt[3]{ -\frac{p}{2} \mp \sqrt{q} }. \] ここで，\(\alpha \beta = 1\) であることは，\(\alpha\) と \(\beta\) が \(x^{3} - 1 = 0\) の解であることから明らかである．なぜなら，\(1, \alpha, \beta\) の積は〔根と係数の関係により〕定数項 \(1\) に等しいからである．したがって， \(\frac{1}{\alpha} = \beta,\quad \frac{1}{\beta} = \alpha\)となり，上記の3つの値は次のように書き換えられる： \[ \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} },\quad \beta \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} },\quad \alpha \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} }. \] したがって，\(x = y + z\) であるから，対応する \(y\) と \(z\) の値を加えると，\(x\) の3つの値は次のようになる： \[ \sqrt[3]{ -\dfrac{p}{2} \pm \sqrt{q} } + \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} }, \] \[ \alpha \sqrt[3]{ -\dfrac{p}{2} \pm \sqrt{q} } + \beta \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} }, \] \[ \beta \sqrt[3]{ -\dfrac{p}{2} \pm \sqrt{q} } + \alpha \sqrt[3]{ -\dfrac{p}{2} \mp \sqrt{q} }. \] ここで，\(\sqrt{q}\) の前にある複号は上の符号をとっても下の符号をとっても，得られる \(x\) の3つの値は同一であることが容易に確認できる．

したがって，平方根 \(\sqrt{q}\) の符号は正負のどちらでもよく，3次方程式の3つの根は，立方根 \(\sqrt[3]{ -\frac{p}{2} \pm \sqrt{q}}\) の3つの値から直ちに得られることになる．

4．任意の3次方程式の解法は6次方程式に帰着されることはすでに示した（第2項）．しかし，次の式 \[ x = \sqrt[3]{ -\dfrac{p}{2} + \sqrt{q} } + \sqrt[3]{ -\dfrac{p}{2} - \sqrt{q} } \] から根号を解消しようとすれば，9次方程式に陥ることになる．まず立方すれば， \[ x^{3} = -p + 3x\, \sqrt[3]{ \dfrac{p^{2}}{4} - q }, \] さらに \(-p\) を左辺に移して再び立方すれば， \[ (x^{3} + p)^{3} = 27\left( \dfrac{p^{2}}{4} - q \right)x^{3}, \] すなわち，\(\frac{p^{2}}{4} - q = -\frac{n^{3}}{27}\) であることから， \[ x^{9} + 3p x^{6} + (3p^{2} + n^{3})x^{3} + p^{3} = 0. \]

ただしこの方程式には，もとの3次方程式 \(x^{3} + nx + p = 0\) の3つの根のほかに，さらに6つの余計な根が含まれていることに注意すべきである．実際，この9次方程式は次の3つの3次方程式に分解される： \begin{align*} &x^{3} + nx + p = 0,\\ &x^{3} + \alpha n x + p = 0,\\ &x^{3} + \beta n x + p = 0. \end{align*} このうち後の2つは，見てのとおり，もとの方程式とは異なる．したがって，この方程式からは，もとの方程式と全く同一の根のみを持つ方程式 \(X^{2} = 0\)の場合（第2項）とは異なり，3次方程式の解法が本来どの次数の方程式に依存しているのかについて何ら確かな結論を導き出すことはできないのである．

5．6次方程式 \[ y^{6} + py^{3} - \dfrac{n^{3}}{27} = 0 \] は3次方程式 \[ x^{3} + nx + p = 0 \] の解法がこの方程式に帰着されることから，簡約方程式と呼ばれる． すでに見たように，3次方程式の根はこの簡約方程式の根に依存している．ここでは逆に，簡約方程式の根が3次方程式の根にどのように依存するかを調べよう．この考察をより一般化し，かつ明晰なものとするために，次のような3次方程式 \[ x^{3} + mx^{2} + nx + p = 0 \] を考え，その根を一般に \(a, b, c\) とする． まず第2項を消去するために，\(x = x' - \frac{m}{3}\) とおく．略記として \[ n' = n - \dfrac{m^{2}}{3},\quad p' = p - \dfrac{mn}{3} + \dfrac{2m^{3}}{27} \] とすれば，変換された方程式は \[ x'^{3} + n' x + p' = 0 \] となる．\(x' = y - \frac{n'}{3y}\) とおけば，簡約方程式は \[ y^{6} + p'y^{3} - \dfrac{n'^{3}}{27} = 0 \] となる． ここで，\(r\) を \[ -\dfrac{p'}{2} + \sqrt{ \dfrac{p'^{2}}{4} + \dfrac{n'^{3}}{27} } \] の立方根とすれば，\(y\) の3つの値は \[ y = r,\quad y = \alpha r,\quad y = \beta r \] であり，対応する \(x'\) の値は \[ x' = r - \dfrac{n'}{3r},\quad x' = \alpha r - \dfrac{n'}{3\alpha r},\quad x' = \beta r - \dfrac{n'}{3\beta r} \] となる．\(x = x' - \frac{m}{3}\) より，\(\frac{n'}{3r} = s\) と略記すれば，\(x\) の3つの値は \[ -\dfrac{m}{3} + r - s,\quad -\dfrac{m}{3} + \alpha r - \dfrac{s}{\alpha},\quad -\dfrac{m}{3} + \beta r - \dfrac{s}{\beta} \] なので， \begin{align*} a &= -\dfrac{m}{3} + r - s,\\[+3pt] b &= -\dfrac{m}{3} + \alpha r - \dfrac{s}{\alpha},\\[+3pt] c &= -\dfrac{m}{3} + \beta r - \dfrac{s}{\beta} \end{align*} となる． このとき， \begin{align*} a - b &= (1 - \alpha)\left( r + \dfrac{s}{\alpha} \right),\\[+3pt] a - c &= (1 - \beta)\left( r + \dfrac{s}{\beta} \right) \end{align*} であるから， \begin{align*} \dfrac{\alpha(a - b)}{1 - \alpha} &= \alpha r + s,\\[+3pt] \dfrac{\beta(a - c)}{1 - \beta} &= \beta r + s. \end{align*} 両辺の差をとって \(\alpha - \beta\) で割れば， \[ r = \dfrac{ \dfrac{\alpha(a - b)}{1 - \alpha} - \dfrac{\beta(a - c)}{1 - \beta} }{ \alpha - \beta }, \] すなわち， \[ r = \dfrac{a}{(1 - \alpha)(1 - \beta)} + \dfrac{\alpha b}{(\alpha - 1)(\alpha - \beta)} + \dfrac{\beta c}{(\beta - 1)(\beta - \alpha)}. \]

ここで \(1, \alpha, \beta\) は方程式 \(x^{3} - 1 = 0\) の3つの根だから， \[ x^{3} - 1 = (x - 1)(x - \alpha)(x - \beta) \] であり，微分をとれば \[ 3x^{2} = (x - \alpha)(x - \beta) + (x - 1)(x - \beta) + (x - 1)(x - \alpha). \] これに \(x = 1, \alpha, \beta\) を代入すれば， \begin{align*} 3 &= (1 - \alpha)(1 - \beta),\\ 3\alpha^{2} &= (\alpha - 1)(\alpha - \beta),\\ 3\beta^{2} &= (\beta - 1)(\beta - \alpha) \end{align*} となるので，これを \(r\) の式に代入すれば， \[ r = \dfrac{a}{3} + \dfrac{b}{3\alpha} + \dfrac{c}{3\beta}, \] あるいは，\(\alpha\beta = 1\) であることから， \[ r = \dfrac{a + \beta b + \alpha c}{3} \] とも書ける． このようにして \(r\) の値，したがって \(y\) の値が得られる．ここで \(\alpha\) と \(\beta\) は互いに入れ替え可能であるから，次の形でもよい： \[ y = \dfrac{a + \alpha b + \beta c}{3}. \]

6．この \(y\) の表現から，簡約方程式が必然的に6次方程式である理由がまずわかる．というのも，この簡約方程式は，もとの方程式の根 \(a, b, c\) に直接依存しているのではなく，それら3つの根が対等に現れる[*1]係数 \(m, n, p\) にのみ依存しているからである．したがって，\(y\) の表現においては，\(a, b, c\) の順序を自由に入れ替えることができなければならない．ゆえに，\(y\) の値は，\(a, b, c\) のあらゆる可能な置換（順列）から得られる異なる値の数だけ存在するはずである．ところで，組合せの理論によれば，3つのものの置換（順列）の数，すなわち異なる並べ方の数は \(3 \cdot 2 \cdot 1\) である．したがって，\(y\) についての簡約方程式もまた \(3 \cdot 2 \cdot 1\) 次，すなわち6次でなければならない．

さらに，\(y\) のこの表現は，簡約方程式が2次方程式のように解ける理由をも明らかにしている．というのも，これは，この方程式に \(y^{3}\) および \(y^{6}\) の冪しか現れないこと（すなわち指数はすべて \(3\) の倍数）に由来することは明らかである．したがって，\(r\) が \(y\) の1つの値であれば，\(\alpha r\) および \(\beta r\) もまたその値でなければならない．なぜなら \(\alpha^{3} = 1\) および \(\beta^{3} = 1\) だからである．そして，これは上で得られた \(y\) の表現において確かに成り立っている． これをより明確に示すために，\(\beta = \alpha^{2}\) であることに注目しよう．実際，\(\alpha \beta = 1\) かつ \(\alpha^{3} = 1\) であるから，\(\alpha \beta = \alpha^{3}\)，すなわち \(\beta = \alpha^{2}\) となる．したがって，\(y\) の表現は次のように書ける： \[ y = \dfrac{a + \alpha b + \alpha^{2} c}{3}. \] ここで，\(a, b, c\) にあらゆる可能な置換を施せば，次の6つの値が得られる： \begin{align*} &\dfrac{a + \alpha b + \alpha^{2} c}{3},\\ &\dfrac{a + \alpha c + \alpha^{2} b}{3},\\ &\dfrac{b + \alpha a + \alpha^{2} c}{3},\\ &\dfrac{b + \alpha c + \alpha^{2} a}{3},\\ &\dfrac{c + \alpha b + \alpha^{2} a}{3},\\ &\dfrac{c + \alpha a + \alpha^{2} b}{3}. \end{align*} これらが簡約方程式の6つの根である． ここで，最初の式に \(\alpha\) を掛け，次いで〔最初の式に〕 \(\beta\)（すなわち \(\alpha^{2}\)）を掛けると，\(\alpha^{3} = 1\) であることから，次の2つの式が得られる： \[ \dfrac{c + \alpha a + \alpha^{2} b}{3},\quad \dfrac{b + \alpha c + \alpha^{2} a}{3}. \] これはそれぞれ第6式と第4式である．同様に，第2式に \(\alpha\) および \(\alpha^{2}\) を掛ければ， \[ \dfrac{b + \alpha a + \alpha^{2} c}{3},\quad \dfrac{c + \alpha b + \alpha^{2} a}{3} \] が得られ，これは第3式と第5式である．同様に，第3式と第4式，あるいは第5式と第6式に \(\alpha\) および \(\alpha^{2}\) を掛けても，他のすべての式が得られることになる．

  1. 対称式の意．

7．これにより，3次方程式の解法が依拠する簡約方程式を直接求める方法が得られる．すなわち， \[ x^{3} + mx^{2} + nx + p = 0 \] を与えられた3次方程式とし，その根を \(a, b, c\) とする．そして，簡約方程式の根が \(a, b, c\) の1次式で表されると仮定し，その一般形を \[ Aa + Bb + Cc \] とする．ここで \(A, B, C\) は \(a, b, c\) に依存しない定数とする． \(a, b, c\) にあらゆる可能な置換を施せば，次の6つの式が得られる： \begin{align*} Aa + Bb + Cc,\\ Aa + Bc + Cb,\\ Ab + Ba + Cc,\\ Ab + Bc + Ca,\\ Ac + Bb + Ca,\\ Ac + Ba + Cb. \end{align*} これらが簡約方程式の6つの根となる． さて，この方程式が \(y^{3}, y^{6}\) のように，指数が3の倍数の冪のみを含む形であるためには，すでに見たように，ある根 \(r\) に対して \(\alpha r\) および \(\beta r\)（すなわち \(\alpha^{2} r\)）もまた根でなければならない．したがって，\(r \)を \[Aa + Bb + Cc\] としたとき， \[\alpha Aa + \alpha Bb + \alpha Cc\] が上記の5つの他の式のいずれかと一致しなければならない． ところが，たとえば \[Aa + Bc + Cb\] または \[Ab + Ba + Cc\] と一致するためには \(\alpha = 1\) でなければならない．なぜなら，前者では \[\alpha A = A,\] 後者では \[\alpha C = C\] となるからである[*1]． しかし，これ〔\(\alpha Aa + \alpha Bb + \alpha Cc\)〕を \[Ab + Bc + Ca\] と比較すれば， \[ \alpha A = C,\quad \alpha B = A,\quad \alpha C = B \] が得られる．これより， \[ C = \alpha A,\quad B = \alpha^{2} A,\quad \alpha^{3} A = A \] となる．よって， \[\alpha^{3} = 1\] なので，\(\alpha\) は方程式 \[x^{3} - 1 = 0\] の根の1つでなければならない． 簡単のため \(A = 1\) とすれば， \[ A = 1,\quad B = \alpha,\quad C = \alpha^{2}\,^{\text{[*2]}} \] となり，これにより，先に得られたものと同じ形の式が，分母の \(3\) を除いて得られることになる．

ここで簡略のため， \begin{align*} r &= a + \alpha b + \alpha^{2} c,\\ s &= a + \alpha c + \alpha^{2} b \end{align*} とおけば，\(r, \alpha r, \alpha^{2} r\) および \(s, \alpha s, \alpha^{2} s\) が変形された方程式（簡約方程式）の6つの根となる． この方程式の未知数を \(y\) とすれば，まず3つの因子 \(y - r, \,y - \alpha r,\, y - \alpha^{2} r\) の積は \(y^{3} - r^{3}\) となり，同様に他の3つの因子の積は \(y^{3} - s^{3}\) となる．したがって，これらすべての積，すなわち簡約方程式 \[ y^{6} - (r^{3} + s^{3}) y^{3} + r^{3} s^{3} = 0 \] と表され，これは求める形である． したがって，あとは \(r^{3} + s^{3}\) および \(r^{3} s^{3}\) の値を求めればよい． \(r\) を3乗し，\(\alpha^{3} = 1\) に注意すれば， \[ r^{3} = a^{3} + b^{3} + c^{3} + 6abc + 3\alpha(a^{2}b + b^{2}c + c^{2}a) + 3\alpha^{2}(ab^{2} + bc^{2} + ca^{2}). \] そして，\(b\) と \(c\) を入れ替えることで， \[ s^{3} = a^{3} + b^{3} + c^{3} + 6abc + 3\alpha(a^{2}c + c^{2}b + b^{2}a) + 3\alpha^{2}(c^{2}a + b^{2}c + a^{2}b). \] 簡略のため， \begin{align*} &a^{3} + b^{3} + c^{3} + 6abc=L,\\ &a^{2}b + b^{2}c + c^{2}a=M,\\ &a^{2}c + b^{2}a + c^{2}b=N \end{align*} とおけば， \begin{align*} r^{3} &= L + 3\alpha M + 3\alpha^{2} N,\\ s^{3} &= L + 3\alpha N + 3\alpha^{2} M. \end{align*} したがって， \[ r^{3} + s^{3} = 2L + 3(\alpha + \alpha^{2})(M + N) \] となるが，\(1, \alpha, \alpha^{2}\) は \(x^{3} - 1 = 0\) の3つの根であり，この方程式には2次の項がないので， \[ 1 + \alpha + \alpha^{2} = 0 \] である．したがって， \[ r^{3} + s^{3} = 2L - 3(M + N). \]

次に，\(r^{3}\) と \(s^{3}\) の積を計算すれば， \[ r^{3} s^{3} = L^{2} + 9(M^{2} + N^{2}) + 3(\alpha + \alpha^{2})[L(M + N) + 3MN]. \] \(\alpha + \alpha^{2} = -1\)なので， \[ r^{3} s^{3} = L[L - 3(M + N)] + 9[(M + N)^{2} - 3MN]. \] さて，量 \(L,\, M + N,\, MN\) は，もとの 3 次方程式の係数 \(m, n, p\) によって決定されるべきものである．しかも，これらは冪根の抽出を伴わずに表される．このことは，これらの量が \(a, b, c\) の各変数について順序をどのように入れ替えても変化しないことから[*3]導かれる．したがって，それぞれの量はただ1つの値しかとり得ないのである．

8．実際，次の関係 \[ -m = a + b + c,\quad n = ab + ac + bc,\quad -p = abc \] があるので，既知の公式により， \[ a^{2} + b^{2} + c^{2} = m^{2} - 2n,\quad a^{3} + b^{3} + c^{3} = -m^{3} + 3mn - 3p \] が得られ，さらに \[ a^{3}b^{3} + a^{3}c^{3} + b^{3}c^{3} = n^{3} - 3mnp + 3p^{2} \] となる．したがって， \begin{align*} &L = -m^{3} + 3mn - 9p,\\　　 &M + N = 3p - mn,\\ &MN = n^{3} + p(m^{3} - 6mn) + 9p^{2} \end{align*} であるから， \begin{align*} &r^{3} + s^{3} = -2m^{3} + 9mn - 27p,\\ &r^{3}s^{3} = m^{6} - 9m^{4}n + 27m^{2}n^{2} - 27n^{3} = (m^{2} - 3n)^{3}. \end{align*} したがって，簡約方程式は \[ y^{6} + (2m^{3} - 9mn + 27p)y^{3} + (m^{2} - 3n)^{3} = 0 \] となり，これは上で（第5項）得られたものと同じである．ただ，ここでの未知数 \(y\) は，前述〔第5項〕の未知数 \(y\) の3倍であることに注意すればよい． この方程式は2次方程式と同様に解くことができる．簡略のため \(y^{3} = z\) とおけば，次のようになる： \[ z^{2} + (2m^{3} - 9mn + 27p)z + (m^{2} - 3n)^{3} = 0. \] この2次方程式の根を \(z', z''\) とすれば， \[ y^{3} = z',\quad y^{3} = z'', \] すなわち， \[ y = \sqrt[3]{z'}\quad \text{または} \quad y = \sqrt[3]{z''} \] となる．\(r\) および \(s\) は \(y\) の2つの値とおいていたので， \begin{align*} r &= a + \alpha b + \alpha^{2} c = \sqrt[3]{z'},\\ s &= a + \alpha c + \alpha^{2} b = \sqrt[3]{z''}. \end{align*} これらの式と \[ a + b + c = -m \] とを組み合わせれば，3つの根 \(a, b, c\) を求めることができる． 実際，\(\alpha^{3} = 1\) および \(1 + \alpha + \alpha^{2} = 0\) であることから， \begin{align*} a &= \dfrac{-m + \sqrt[3]{z'} + \sqrt[3]{z''}}{3},\\[+3pt] b &= \dfrac{-m + \alpha^{2} \sqrt[3]{z'} + \alpha \sqrt[3]{z''}}{3},\\[+3pt] c &= \dfrac{-m + \alpha \sqrt[3]{z'} + \alpha^{2} \sqrt[3]{z''}}{3} \end{align*} となり，これは既に得た結果と一致する．

9．この論点をさらに明確にするために，次の点にも注意しておくとよい．すなわち，第7項で導入された量 \(M\) および \(N\) は，3つの根 \(a, b, c\) の間に任意の置換を施せば，互いに入れ替わる関係にある．したがって，\(M\) と \(N\) は2次方程式の根であるに違いない． 実際，この方程式の未知数を \(t\) とおけば，その〔方程式の〕形式は必然的に \[ t^{2} - (M + N)t + MN = 0 \] となる．ここで第8項で得られた \(M + N\) および \(MN\) の値を代入すれば， \[ t^{2} - (3p - mn)t + n^{3} + (m^{3} - 6mn)p + 9p^{2} = 0. \] したがって，この2次方程式を解けば，\(M\) および \(N\) の値が得られる． また，量 \(L\) はすでに \[ L = -m^{3} + 3mn - 9p \] として与えられているので，第7項の式により \(r^{3}\) および \(s^{3}\) の値，あるいは第8項の式により \(z', z''\) の値が得られる．それらは次のようになる： \begin{align*} z' &= L + 3\alpha M + 3\alpha^{2} N,\\ z'' &= L + 3\alpha N + 3\alpha^{2} M. \end{align*} これらの値を用いれば，先ほど見たように，3つの根 \(a, b, c\) の値も得られることになる．

さらに，\(M\) と \(N\) のこのような性質から， \[ z = y^{3} = (a + \alpha b + \alpha^{2} c)^{3} \] という量が2次方程式のみに依存している理由が明白となる．したがって，\(y\) についての方程式には \(y^{3}\) および \(y^{6}\) の冪しか現れないのである．

10．これまでに見てきた3次方程式の解法は，一般にカルダノの公式と呼ばれており，解析学者たちが知っている唯一の方法である．しかし，この他にも，チルンハウス氏による別の方法が存在する．この方法はカルダノのものほど単純ではないが，より直接的かつ一般的であるという利点をもっている． この方法は 1683 年の『学術報告誌（Acta Eruditorum）』に掲載されており，与えられた方程式から中間項を望む個数だけ消去することを目的としている．著者はこの方法をその目的に対して一般的なものとして提示しており，実際にそれが可能であることを我々は見ることになる．ただし，この方法はしばしば，もとの方程式よりも高次の方程式を解くことを必要とするため，4次以上の方程式には適用が困難である．

チルンハウス氏は次のように指摘している：未知数 \(x\) を含む方程式において， \[x = y + a\] とおけば，任意の項を消去することができる．ここで \(y\) は新たな未知数であり，\(a\) は未定の定数である．同様に，たとえば， \[ x^{2} = bx + a + y \] とおけば，任意の2項を消去することができ，また， \[ x^{3} = cx^{2} + bx + a + y \] とおけば，任意の3項を消去することができる．以降も同様で，\(a, b, c, \dots\) はすべて未定係数であり，その個数は消去したい項の数と同数にする．これは，未知数の個数を，満たすべき条件の数と一致させるためである．

したがって，仮定した新たな方程式を用いて，もとの方程式から未知数 \(x\) を消去すれば，\(y\) についての方程式が得られる．この方程式はもとの方程式と同じ次数をもち，かつ，未定係数 \(a, b, c, \dots\) の数だけ項を消去することができる．

そこで，3次方程式 \[ x^{3} + mx^{2} + nx + p = 0 \] を取り，2つの中間項を消去できるように， \[ x^{2} = bx + a + y \] と仮定する．このとき， \[ x^{3} = bx^{2} + ax + yx \] となるが，\(x^{2}\) の値を代入すれば， \[ x^{3} = (b^{2} + a + y)x + b(a + y) \] となる．これらをもとの方程式に代入すれば， \[ (b^{2} + mb + n + a + y)x + (b + m)(a + y) + p = 0 \tag{A} \] となり，ここから \(x\) の値は \[ x = -\dfrac{(b + m)(a + y) + p}{b^{2} + mb + n + a + y} \] と求められる．この値を \[x^{2} = bx + a + y\] に代入すれば，次の式が得られる．簡略のため， \(b + m = c\), \(b^{2} + mb + n = d\) とおくと， \[ [c(a + y) + p]^{2} + b[c(a + y) + p](d + a + y) - (a + y)(d + a + y)^{2} = 0. \] \(a + y\) の冪に従って整理し，\(c, d\) の値を戻せば， \[\left\{\begin{aligned} &(y + a)^{3} - (mb + m^{2} - 2n)(y + a)^{2} \\ &\quad + [nb^{2} + (mn - 3p)b + n^{2} - 2mp](y + a) - p(b^{3} + mb^{2} + nb + p) = 0. \end{aligned} \right.\tag{B}\] \(y+a\)の冪を展開すれば，方程式 \[ y^{3} + Ay^{2} + By + C = 0 \] が得られ，その係数は次のようになる： \[ \begin{aligned} A &= 3a - mb - m^{2} + 2n,\\[+3pt] B &= 3a^{2} - 2a(mb + m^{2} - 2n) + nb^{2} + (mn - 3p)b + n^{2} - 2mp,\\[+3pt] C &= a^{3} - (mb + m^{2} - 2n)a^{2} + [nb^{2} + (mn - 3p)b + n^{2} - 2mp]a \\ &\quad - p(b^{3} + mb^{2} + nb + p). \end{aligned} \] ここで，\(A = 0\) および \(B = 0\) とおけば，第2項と第3項が消去され，次の2つの方程式が得られる： \[ \begin{aligned} &3a - mb - m^{2} + 2n = 0,\\ &3a^{2} - 2a(mb + m^{2} - 2n) + nb^{2} + (mn - 3p)b + n^{2} - 2mp = 0. \end{aligned} \] これにより \(a\) および \(b\) を定めることができ，\(y\) についての方程式は \[ y^{3} + C = 0 \] の形に簡約される．この方程式は直ちに次の3つの根を与える： \[ y = -\sqrt[3]{C},\quad y = -\alpha \sqrt[3]{C},\quad y = -\alpha^{2} \sqrt[3]{C}, \] ここで \(1, \alpha, \alpha^{2}\) は方程式 \(x^{3} - 1 = 0\) の3つの根である． したがって，まず，上で得られた \(x\) の式に，先の方程式から定まる \(a, b\) の値を代入する．そして \(y\) の3つの根を代入すれば，もとの3次方程式の3つの根 \(x\) が直ちに得られることになる．

さて，\(a\) および \(b\) を決定するための2つの方程式のうち，前者は1次方程式，後者は2次方程式であるから，これらの量の決定は \(1 \cdot 2\) 次[*1]，すなわち2次方程式に帰着されることが明らかである． 実際，まず \[ a = \dfrac{mb + m^{2} - 2n}{3} \] となり，この値を2番目の方程式に代入すれば， \[ (m^{2} - 3n)b^{2} + (2m^{3} - 7mn + 9p)b + m^{4} - 4m^{2}n + 6mp + n^{2} = 0 \] が得られる．この2次方程式からは \(b\) の値が2つ得られるが，それらはいずれも \(x\) に対して同じ値を与えるので，どちらを用いても差し支えない．

この方法は，通常の方法では6次の簡約方程式に至るのに対し，直接的に2次の簡約方程式に導くという利点をもっている． しかしながら，この解法もカルダノの解法において指摘したのと同様の難点を免れているわけではない．すなわち，この解法は本質的には3次方程式の解法というよりも，6次方程式の解法となっているという点である（第2項）． 実際，\(y\) が3つの値をもち，\(b\)（および \(a\)）が2つの値を持つ以上，\(x\) の値は \(6\) 通り存在することになる．したがって，それらは6次方程式の根であると考えざるを得ない． もっとも，それぞれが重根であるため，これら6つの値は3つに数を減らすことは容易に証明できるし，カルダノの公式についてすでにそれを確認している．

  1. \((1\times2)\) 次の意．\(k\) 個の未知数をもつ次数 \(n_1, n_2,\dots n_k\) の連立方程式から消去により未知数が1つだけの方程式をつくると，一般には，\(n_1n_2\cdots n_k\) 次の方程式が得られる．これを既知としている．

11．チルンハウス氏の方法について，重要な注意点がある．すなわち，\(a\), \(b\), \(y\) の値が得られたからといって，\(x\) の値として方程式 \[ x^{2} - bx - a - y = 0 \] の〔2つの〕根のどちらを選んでもよい――著者〔チルンハウス〕はそうしている――としてはいけない．それが許されるためには，この方程式がもとの方程式の根のうち2つを含んでいなければならず，したがって \(b\) はその2つの根の和でなければならない． ところが，\(b\) が3つの根のうちどの2つの和であるべきかについて，特別な理由は存在しない．したがって，\(b\) は3つの根から2つを選ぶすべての組合せに対応するだけの値をもたなければならず，その数は \(\frac{3 \cdot 2}{2} = 3\) である． ところが，すでに見たように，\(b\) は2次方程式に従う量であり，したがってその値は2つしか存在しない．

この方法の本質は，仮定された方程式がもとの方程式と共通の根をもつようにすることにある．したがって，\(a\), \(b\), \(y\) の値をこの条件が成り立つように定めたならば，\(x\) の値としては，方程式 \[ x^{2} - bx - a - y = 0 \] を満たす根のうち，与えられた方程式 \[ x^{3} + mx^{2} + nx + p = 0 \] との共通の根を選ばなければならない． そのためには，この2つの方程式の最大公約式を求めればよい．この最大公約式は \(x\) に関して1次式であり，それが \(x\) の値を与える．この値は，与えられた方程式の根の1つでもある． そして，この \(x\) の値は，我々が2つの方程式から高次の \(x\) を順に消去して得たものと一致することは，容易に理解できる．

実際，通常の消去法とは，同じ未知数が任意の冪に累乗されている2つの与えられた方程式から，未知数の最高次の項を順次消去していくことによって，未知数の最高次が次第に低くなる一連の方程式を導き出し，最終的に未知数を含まない方程式に至るという方法であり，それが消去の結果[*1]である．

この方法は本質的には，2つの与えられた方程式の左辺をなす2つの式[*2]の最大公約式を求める方法と同じである．すなわち，必要な除法を逐次行い，得られる余りを 0 に等しいとおけば，それらは消去によって得られる一連の方程式と同じである． 未知数を含まない最後の余りが 0 になることは，2つの与えられた式が1次の共通因数をもつための条件であり，その直前の余り（未知数が1次で現れるもの）が，その共通因数である．したがって，この1次の余りを 0 に等しいとおけば，未知数の値が得られ，それは2つの方程式に共通する根となる．

第10項の例においては，方程式 (A) および (B) が，それぞれこの「直前の余り」と「最後の余り」に対応している．したがって，方程式 (A) から得られる \(x\) の値こそが，与えられた方程式の根となりうる唯一の値である．

12．この指摘に関連して，さらに次のような注意を加えておくべきであろう．すなわち，2つの方程式が複数の共通根をもつようにするにはどうすればよいかという問題である． 明らかに，2つの方程式が2つの共通根をもつためには，それらがある2次の因数で割り切れなくてはならない．したがって，2つの与えられた方程式の左辺を構成する式の最大公約式を求める際，未知数が2次で現れる余りに到達した時点で，その次の余りが恒等的に 0 になる必要がある． この最後の余りは，未知数を含まない項と，未知数が1次で現れる項の2つから成る．ゆえに，それらの各項を個別に 0 に等しいとおけば，2つの与えられた方程式が2つの共通根をもつための必要条件を与える2つの方程式が得られる． これは，消去法を用いる場合も同様である．このときは，未知数が1次で現れる式に至ったところで停止し，この1次の方程式を未知数とは無関係に，式を構成する2つの項それぞれを個別に 0 に等しくすればよい．この操作によって，その直前の2次方程式が，2つの与えられた方程式に共通する2つの根を含むことになる．

このようにして，2つの方程式が3つ以上の共通根をもつための条件を求める方法も明らかとなる．しかし，1つの共通根をもつための条件が得られさえすれば，そこから2つ以上の共通根をもつための条件は容易に導くことができる．

このために，未知数 \(x\) を含む2つの与えられた方程式を一般に \[ {P} = 0 \quad \text{および} \quad {Q} = 0 \] と表すことにする．ここで，\({P} = 0\) の代わりに \({P} = y\) とし，この2つの方程式から \(x\) を消去すれば，\(y\) についての方程式 \[ y^{m} + ay^{m-1} + \cdots + py^{2} + qy + r = 0 \] が得られる． さて，\({P} = 0\) および \({Q} = 0\) が共通の根をもつためには，\(y = 0\) がこの方程式の解でなければならない．したがって， \[r = 0\] が1つの共通根をもつための必要条件となる． しかし，2つの方程式が2つの共通根をもつようにしたい場合には，\(y\) が2つの値に対して \(0\) になることが必要である． したがって， \[r = 0 \quad\text{および}\quad q = 0\] の2つの条件[*1]が導かれる．同様に，3つの共通根をもつには， \[r = 0, \quad q = 0 \quad\text{および}\quad p = 0\] の3つの条件が必要となり，…等々以下同様である．

ここで注意すべきは，\({P} = 0\) を \({P} = y\)，すなわち \({P} - y = 0\) に変えるには，\({P} = 0\) の定数項から \(y\) を引けばよいということである．すなわち， \[{P} = x^{n} + \alpha x^{n-1} + \cdots + \rho\] とすれば，\(\rho\) を \(\rho - y\) に置き換えればよい． 方程式 \[ y^{m} + ay^{m-1} + \cdots + py^{2} + qy + r = 0 \] は，\({P} = y\) および \({Q} = 0\)から \(x\) を消去して得られるものである（仮定による）．したがって，\(y = 0\) を代入すれば，\(r = 0\) は \({P} = 0\) および \({Q} = 0\) から \(x\) を消去して得られる方程式となる． よって，この \(r = 0\) において \(\rho\) を \(\rho - y\) に置き換えれば，直ちに \[ y^{m} + ay^{m-1} + \cdots + py^{2} + qy + r = 0 \] が得られる． ところで，\(r\) が \(\rho\) の関数であるとき，\(\rho\) を \(\rho - y\) に置き換えた式は，微分法を用いて次のように展開できる： \[ r - \frac{dr}{d\rho}y + \frac{1}{2} \frac{d^{2}r}{d\rho^{2}} y^{2} - \frac{1}{2 \cdot 3} \frac{d^{3}r}{d\rho^{3}} y^{3} + \dots. \] したがって，各項を比較すれば， \[ q = -\frac{dr}{d\rho}, \quad p = \frac{1}{2} \frac{d^{2}r}{d\rho^{2}}, \dots. \] 以上から次の結論が得られる．もし， \[r = 0\] が \({P} = 0\) と \({Q} = 0\) が1つの共通根をもつための必要条件であるならば，2つの共通根をもつための条件は \[ r = 0 \quad \text{および} \quad \frac{dr}{d\rho} = 0 \] であり，3つの共通根をもつための条件は \[ r = 0, \quad \frac{dr}{d\rho} = 0, \quad \frac{d^{2}r}{d\rho^{2}} = 0 \] であり，…等々以下同様である．ここで \(\rho\) は与えられた方程式のうちの1つの定数項である．

14．以上より一般に次のことが結論される．すなわち，チルンハウス氏の方法に従えば，変数 \(y\) についての変換方程式は，もとの方程式と同じ次数をもち，その中に現れる \(y, a, b, c, \dots\)（および \(x\) の最高次の係数である 1 を含む）は，すべて同じ次数の積として現れる．すなわち，もとの方程式の次数と同じだけの次元をもつ積である．[*1]

たとえば，未知数 \(x\) についてのもとの方程式が次数 \(m\) であり，補助方程式として \[ y + a + bx + cx^{2} + \cdots = x^{r} \] の形をとるとすると，\(y\) についての変換方程式はやはり次数 \(m\) であり，その形は \[ y^{m} + {A} y^{m-1} + {B} y^{m-2} + {C} y^{m-3} + \cdots = 0 \] と表される．ここで，係数 \({A}\) は \(a, b, c, \dots\) の1次の関数，\({B}\) はそれらの2次の関数，\({C}\) は3 次の関数，…等々以下同様となる．

一般に，第 \(n\) 項の係数は，\(a, b, c, \dots\) についての \(n-1\) 次の多項式となる． したがって，消去したい項の数だけ \(a, b, c, \dots\) のような未定係数を導入すれば，たとえば第2項を消去するには，1つの未知数についての1次方程式を解くだけでよい．第2項と第3項を同時に消去するには，2つの未知数についての2つの方程式（1次と2次）を解く必要があり，その結果として得られる最終方程式は2次となる（これはすでに上で見た）． さらに第2項，第3項，第4項を消去するには，3つの未知数についての3つの方程式（1次，2次，3次）を解く必要があり，最終的な方程式は一般に\(1 \cdot 2 \cdot 3 = 6\)次となる．

一般に，第 \(p\) 項，第 \(q\) 項，第 \(r\) 項，…を同時に消去するには，それらの項の数だけの方程式を，同数の未知数について解く必要があり，それらの方程式の次数はそれぞれ \(p-1\), \(q-1\), \(r-1\), … となる．したがって，最終的な方程式は一般に\((p - 1)(q - 1)(r - 1)\)次となる． したがって，この方法によって変換方程式 \[ y^{m} + {A} y^{m-1} + {B} y^{m-2} + \cdots + {M} = 0 \] の中間項をすべて消去して，それを \(y^{m} + {M} = 0\) という常に解ける形に帰着させようとすれば，一般に \(1 \cdot 2 \cdot 3 \cdots (m - 1)\) 次の方程式に行き着くことになる．この次数は，\(m = 3\) の場合を除けば，もとの方程式の次数 \(m\) を常に上回ることになる．

15．さて，チルンハウス氏の方法によって得られた3次方程式の解法に立ち返り，先に説明した消去の手続きに依らず，そしてアプリオリに，この方法がなぜ2次の簡約方程式に直接導くのかを考察しよう． このために，補助方程式として \[ x^{2} = bx + a + y \] を考える．ここで \(y\) は，2項からなる3次方程式 \[ y^{3} + {C} = 0 \] によって定まるものとする．この方程式の根は \[ y = -\sqrt[3]{{C}}, \quad y = -\alpha \sqrt[3]{{C}}, \quad y = -\alpha^{2} \sqrt[3]{{C}} \] である． この3つの根は，もとの方程式 \[ x^{3} + mx^{2} + nx + p = 0 \] の3つの根に対応するものでなければならない．これらの根を \(x', x'', x'''\) とすれば，次の3つの等式が得られる： \[\left\{ \begin{aligned} x'^{2} &= bx' + a - \sqrt[3]{{C}}, \\ x''^{2} &= bx'' + a - \alpha \sqrt[3]{{C}}, \\ x'''^{2} &= bx''' + a - \alpha^{2} \sqrt[3]{{C}}. \end{aligned}\right. \tag{C} \] ここから，\(\sqrt[3]{{C}}\) を消去すれば，\(a\) および \(b\) の値を求めることができる．そのためには，2番目の式を \(\alpha\) 倍，3番目の式を \(\alpha^{2}\) 倍してから，3式を加えればよい．すでに見たように \(\alpha^{4} = \alpha\) および \(1 + \alpha + \alpha^{2} = 0\) が成り立つので，次の等式を得る： \[ x'^{2} + \alpha x''^{2} + \alpha^{2} x'''^{2} = b(x' + \alpha x'' + \alpha^{2} x'''), \] したがって， \[ b = \frac{x'^{2} + \alpha x''^{2} + \alpha^{2} x'''^{2}}{x' + \alpha x'' + \alpha^{2} x'''}. \]

この \(b\) の式から，\(b\) を定める方程式の次数を直ちに予見することができる．実際，この方程式は，\(b\) が取りうる値の個数と同じだけの根をもつはずである．ところで，\(b\) の異なる値は，\(x', x'', x'''\) の間の置換によってのみ生じうる[*1]．これらの置換は全部で6通りある（第6項以降を参照．そこでの \(a, b, c\) はここでの \(x', x'', x'''\) に対応する）．したがって，\(b\) は次の6つの値をとりうる： \[ \begin{aligned} &\frac{x'^{2} + \alpha x''^{2} + \alpha^{2} x'''^{2}}{x' + \alpha x'' + \alpha^{2} x'''}, \\[+5pt] &\frac{x'^{2} + \alpha x'''^{2} + \alpha^{2} x''^{2}}{x' + \alpha x''' + \alpha^{2} x''}, \\[+5pt] &\frac{x''^{2} + \alpha x'''^{2} + \alpha^{2} x'^{2}}{x'' + \alpha x''' + \alpha^{2} x'}, \\[+5pt] &\frac{x''^{2} + \alpha x'^{2} + \alpha^{2} x'''^{2}}{x'' + \alpha x' + \alpha^{2} x'''}, \\[+5pt] &\frac{x'''^{2} + \alpha x'^{2} + \alpha^{2} x''^{2}}{x''' + \alpha x' + \alpha^{2} x''}, \\[+5pt] &\frac{x'''^{2} + \alpha x''^{2} + \alpha^{2} x'^{2}}{x''' + \alpha x'' + \alpha^{2} x'}. \end{aligned} \] したがって，一般的に言えば，\(b\) についての方程式は6次になるはずである． しかし，これら6つの値のうち，第1，第3，第5の各式は互いに等しく，また第2，第4，第6の各式も互いに等しいことに注目されたい．実際，第1式の分子と分母を \(\alpha\) 倍しても値は変わらず，\(\alpha^{3} = 1\), \(\alpha^{4} = \alpha\) であることから，第5式と一致する．また，\(\alpha^{2}\) 倍すれば第3式になる．同様に，第2式の分子と分母を \(\alpha\) 倍すれば第4式に，\(\alpha^{2}\) 倍すれば第6式になる． したがって，\(b\) についての6次方程式は，3つの等しい根と，他の3つの等しい根をもつことになり，それは2次方程式の立方[*2]に他ならない．ゆえに，\(b\) は2次方程式によって与えられることになる．これはすでに第10項で見たとおりである． 次に \(a\) について考えると，3つの等式 (C) を加えることで，\(1 + \alpha + \alpha^{2} = 0\) により，次の式が得られる： \[ x'^{2} + x''^{2} + x'''^{2} = b(x' + x'' + x''') + 3a. \] ところで， \[ x' + x'' + x''' = -m, \quad x'^{2} + x''^{2} + x'''^{2} = m^{2} - 2n \] であるから， \[ m^{2} - 2n = -bm + 3a, \] すなわち， \[ a = \frac{bm + m^{2} - 2n}{3}. \] したがって，\(b\) の値がわかれば，\(a\) の値も直ちに求まる．

さらに，2つの分母 \[ x' + \alpha x'' + \alpha^{2} x'''\quad\text{と}\quad x' + \alpha x''' + \alpha^{2} x'' \] を掛け合わせると，次のようになる： \[ x'^{2} + x''^{2} + x'''^{2} + (\alpha + \alpha^{2})(x'x'' + x'x''' + x''x'''). \] ここで， \[ x'^{2} + x''^{2} + x'''^{2} = m^{2} - 2n, \quad x'x'' + x'x''' + x''x''' = n, \quad \alpha + \alpha^{2} = -1 \] であるから，積は \(m^{2} - 3n\) となる． また，分子 \[x'^{2} + \alpha x''^{2} + \alpha^{2} x'''^{2}\] に，分母 \[x' + \alpha x''' + \alpha^{2} x''\] を掛けると，次の式が得られる： \[ x'^{3} + x''^{3} + x'''^{3} + \alpha(x'^{2}x''' + x''^{2}x' + x'''^{2}x'') + \alpha^{2}(x'^{2}x'' + x''^{2}x''' + x'''^{2}x'). \] これは（\(x', x'', x'''\) は，以前の \(a, b, c\) と同じなので)， \[L-6x' x''x'''+\alpha M+\alpha^2 N\] となる（第7項）．\(x'x''x''' = -p\)なので，これは， \[ {L} + 6p + \alpha {M} + \alpha^{2} {N} \] に等しい． したがって，分数 \[ \frac{x'^{2} + \alpha x''^{2} + \alpha^{2} x'''^{2}}{x' + \alpha x'' + \alpha^{2} x'''} \] は次のように書き換えられる： \[ \frac{{L} + 6p + \alpha {M} + \alpha^{2} {N}}{m^{2} - 3n}. \] 同様に，もう一方の分数 \[ \frac{x'^{2} + \alpha x'''^{2} + \alpha^{2} x''^{2}}{x' + \alpha x''' + \alpha^{2} x''} \] は，分子と分母に \(x'+\alpha x''+\alpha^2 x'''\) を掛けることで， \[ \frac{{L} + 6p + \alpha {N} + \alpha^{2} {M}}{m^{2} - 3n} \] となる． ところで，第7項では， \[ r^{3} = {L} + 3\alpha {M} + 3\alpha^{2} {N}, \quad s^{3} = {L} + 3\alpha {N} + 3\alpha^{2} {M} \] であったから， \[ \alpha {M} + \alpha^{2} {N} = \frac{r^{3} - {L}}{3}, \quad \alpha {N} + \alpha^{2} {M} = \frac{s^{3} - {L}}{3}. \] したがって，2つの分数はそれぞれ， \[ \frac{r^{3} + 2{L} + 18p}{3(m^{2} - 3n)}, \quad \frac{s^{3} + 2{L} + 18p}{3(m^{2} - 3n)}, \] あるいは（第8項）， \[ \frac{z' + 2{L} + 18p}{3(m^{2} - 3n)}, \quad \frac{z'' + 2{L} + 18p}{3(m^{2} - 3n)} \] と書ける．ここで \(z', z''\) は，次の方程式の根である： \[ z^{2} + (2m^{3} - 9mn + 27p)z + (m^{2} - 3n)^{3} = 0. \] これは，カルダノの方法によって得られる簡約方程式である．

17．第10項でチルンハウス氏の方法により得られた \(x\) の式は，明らかに次の形に書き換えることができる： \[ x = \frac{f + gy}{k + y}. \] ここで \(f, g, k\) は未定係数，\(y\) は2項からなる3次方程式 \[ y^{3} + h = 0 \] の根である． したがって，この2つの方程式を用いて \(y\) を消去すればよい．第1の式からは \[ y = \frac{f - kx}{x - g} \] が得られ，これを第2の方程式に代入すれば， \[ h + \left( \frac{f - kx}{x - g} \right)^{3} = 0 \] という3次方程式が得られる．この方程式をもとの方程式と比較することで，\(f, g, k, h\) の値（このうち1つは任意に定めることができる）が決定される． 3次方程式のこの解法は，すでにベズー氏がこの主題に関して1762年に『パリ科学アカデミー紀要（Mémoires de l'Académie des Sciences de Paris）』に寄稿した優れた論文の中で用いており，その中で著者はこの種の代入法を用いてあらゆる次数の広範な種類の方程式を解くことに成功している． ここでは，このような代入法によって3次方程式を解く際に，あらかじめどのような結果が予見されるかを知るためには，\(f\) や \(g\) などの係数のいずれかを定める方程式の次数と形をアプリオリに求めればよい，ということだけを指摘しておくにとどめる． そのためには，\(y^{3} + h = 0\) であることから，\(y\) の値は \(-\sqrt[3]{h}, \, -\alpha \sqrt[3]{h},\, -\alpha^{2} \sqrt[3]{h}\) の3つであることに注目する．これらを \(x\) の式 \[ x = \frac{f + gy}{k + y} \] に代入すれば，\(x\) の3つの値 \(x', x'', x'''\) が得られる．

したがって，方程式 \[ x(k + y) = f + gy, \] すなわち， \[ kx - f + (x - g)y = 0 \] を用いれば，次の3つの式が得られる： \begin{align*} &kx' - f - (x' - g)\sqrt[3]{h} = 0, \\ &kx'' - f - \alpha(x'' - g)\sqrt[3]{h} = 0, \\ &kx''' - f - \alpha^{2}(x''' - g)\sqrt[3]{h} = 0. \end{align*} これらをまず加えると，\(x' + x'' + x''' = -m\) および \(1 + \alpha + \alpha^{2} = 0\) であることから， \[ mk + 3f + (x' + \alpha x'' + \alpha^{2} x''')\sqrt[3]{h} = 0 \] が得られる． さらに，第2式に \(\alpha\) を，第3式に \(\alpha^{2}\) をそれぞれ乗じてから3つの式を加えると， \[ k(x' + \alpha x'' + \alpha^{2} x''') - (x' + \alpha^{2} x'' + \alpha x''')\sqrt[3]{h} = 0 \] となる． ここから， \[ \sqrt[3]{h} = \frac{k(x' + \alpha x'' + \alpha^{2} x''')}{x' + \alpha x''' + \alpha^{2} x''} \] が得られ，これを先の式に代入して \(k\) で割ると， \[ m + \frac{3f}{k} + \frac{(x' + \alpha x'' + \alpha^{2} x''')^{2}}{x' + \alpha x''' + \alpha^{2} x''} = 0, \] すなわち， \[ \frac{f}{k} = -\frac{m}{3} - \frac{(x' + \alpha x'' + \alpha^{2} x''')^{2}}{3(x' + \alpha x''' + \alpha^{2} x'')} \] が得られる． この式から，\(\frac{f}{k}\) は，2つの異なる値しか取り得ないこと，したがって，2次方程式によって定まることがわかる．なぜなら，分数 \[ \frac{(x' + \alpha x'' + \alpha^{2} x''')^{2}}{x' + \alpha x''' + \alpha^{2} x''} \] は，\(x', x'', x'''\) の置換によって，そのままか，次の分数に変わるかのいずれかだからである： \[ \frac{(x' + \alpha x''' + \alpha^{2} x'')^{2}}{x' + \alpha x'' + \alpha^{2} x'''}. \] このことは，はじめの分数の分子と分母に \[x' + \alpha x'' + \alpha^{2} x'''\] を掛け，2番目の分数の分子と分母に \[x' + \alpha x''' + \alpha^{2} x''\] を掛けることで明確になる．なぜなら，それらは次のようになる（第16項）： \[ \frac{(x' + \alpha x'' + \alpha^{2} x''')^{3}}{m^{2} - 3n}\quad\text{と}\quad \frac{(x' + \alpha x''' + \alpha^{2} x'')^{3}}{m^{2} - 3n}, \] すなわち， \[ \frac{r^{3}}{m^{2} - 3n}\quad\text{と} \quad \frac{s^{3}}{m^{2} - 3n}, \] あるいは， \[ \frac{z'}{m^{2} - 3n}\quad\text{と}\quad \frac{z''}{m^{2} - 3n}. \] したがって，\(\frac{f}{k}\) の2つの値は， \[ -\frac{m}{3} - \frac{z'}{3(m^{2} - 3n)}\quad\text{と}\quad -\frac{m}{3} - \frac{z''}{3(m^{2} - 3n)} \] であり，ここで \(z', z''\) は前述の \(z\) についての方程式の根である．

18．再び\(x\) の式 \[ x = \frac{f + gy}{k + y} \] を取り上げよう．ここで \(y\) は，方程式 \(y^{3} + h = 0\) によって与えられる根である．この分数の分母 \(k + y\) に含まれる冪根を解消するために，分子と分母の両方に \(k^{2} - ky + y^{2}\) を掛けると，次のようになる： \[ \frac{k^{2}f + (k^{2}g - kf)y + (f - kg)y^{2} + gy^{3}}{k^{3} + y^{3}}. \] \(y^{3}\) に\(-h\) を代入することで，この式は \[ \frac{k^{2}f - hg + (k^{2}g - kf)y + (f - kg)y^{2}}{k^{3} - h} \] となる．これは，より簡潔に \[ a + by + cy^{2} \] の形に整理できる．したがって，一般に \[ x = a + by + cy^{2} \] と書ける．ここで \(a, b, c\) は未定係数，\(y\) は2項からなる3次方程式 \(y^{3} + h = 0\) の根である．

この \(x\) の表現は，オイラー氏およびベズー氏が3次方程式の根を表すために採用したものであり，これらの著者はこの形式を一般に任意次数の方程式にも拡張できると考えている．この点については，『ペテルブルク新報告（Nouveaux Commentaires de Pétersbourg）』第IX巻および1765年の『パリ科学アカデミー紀要（Mémoires de l'Académie des Sciences de Paris）』を参照されたい．

したがって，この方法に従って3次方程式を解くには，次の2つの方程式 \[ x = a + by + cy^{2} \quad\text{と}\quad y^{3} + h = 0 \] を用いて \(y\) を消去すればよい．すると，\(x\) についての3次方程式が得られる．これは，上で述べた消去理論によっても確認できる．この方程式をもとの方程式と項ごとに比較すれば，\(a, b, c, h\) のうち3つを決めるための3つの方程式が得られ，残る1つは任意に定めることができる． ベズー氏は，簡便さを優先し最初から \(h = -1\) と置いている．対してオイラー氏は計算の過程においてすべての未定係数を保持し，最終的に最も簡潔な結果が得られるように，適切なものを1に定める．この点が，両者の手法の唯一の違いである．

20．以上が，今日までに見出された3次方程式の主要な解法の全容である． 我々がこれまでに行ってきた分析が物語るように，これらの手法は根底においてすべて同一のものである．なぜなら，それらはいずれも， \(x' + \alpha x'' + \alpha^{2} x'''\) あるいは\(\left(x' + \alpha x'' + \alpha^{2} x'''\right)^{3}\)，もしくはこれらと比例関係にある量を根をもつ簡約方程式を見出すことに帰着するからである． 簡約方程式の根が \(x' + \alpha x'' + \alpha^{2} x'''\) の場合，その簡約方程式は6次であるが，未知数の3乗および6乗の項しか含まないため，2次方程式に帰着させて解くことができる．この理由については第6項で述べたとおりである． 一方，簡約方程式の根が \(\left(x' + \alpha x'' + \alpha^{2} x'''\right)^{3}\) の場合，この簡約方程式は2次にしかなり得ない．これは前者の帰結から自明であるのみならず，我々はすでに直接的な証明も与えている（第9項）．

21．この節を終える前に，方程式 \[ x^{3} - 1 = 0 \] の解法について一言述べ，あわせて一般の方程式 \[ x^{n} - 1 = 0 \] の解法についていくつかの注釈を付しておこう．これらは後に役立つことがあるだろう．

まず明らかなように，\(x^{3} - 1 = 0\) の1つの根は \(1\) である．したがって，他の2つの根を求めるには，この方程式を \(x - 1\) で割ればよく，得られるのは \[ x^{2} - x + 1 = 0\,\,^{\text{[*1]}}\] であり，これより， \[ x = \frac{-1 \pm \sqrt{-3}}{2}. \]

したがって， \[ \alpha = \frac{-1 + \sqrt{-3}}{2}, \quad \beta = \frac{-1 - \sqrt{-3}}{2} \] である．そして，\(\beta = \alpha^{2}\) であることは，すでにアプリオリに示しているが，実際に \(\alpha\) を2乗することで確かめられる： \[ \frac{1 - 2\sqrt{-3} - 3}{4} = \frac{-1 - \sqrt{-3}}{2} = \beta. \]

一般に，2項からなる方程式 \[ x^{n} - 1 = 0 \] を考えると，\(n\) が合成数であって \(n = pq\) の形に書けるとき，この方程式の解法は，次数 \(p\) および次数 \(q\) の2つの方程式の解法へと還元される． 実際，\(x^{q} = y\) とおけば，\(x^{n} = y^{p}\) となり，したがって， \[ y^{p} - 1 = 0 \] が得られる．この \(p\) 次方程式を解いて，その1つの根を \(\alpha\) とすれば， \[ x^{q} - \alpha = 0 \] を解けばよく，\(x = t \sqrt[q]{\alpha}\) とおけば， \[ t^{q} - 1 = 0 \] となる．この新たな方程式を解けば \(t\) の値が得られ，したがって \(x\) の値も得られる．

以上から明らかなように，\(x^{n} - 1 = 0\) の解法は，\(n\) が合成数である場合には，\(n\) の素因数の個数だけ同様の方程式を解くことに帰着され，それぞれの方程式の次数は \(n\) の素因数に等しい．

したがって，真に困難なのは，\(n\) が素数である場合の \(x^{n} - 1 = 0\) の解法である．

一般に，\(n\) が奇数である場合，すなわち \(n = 2p + 1\) のとき，方程式 \[ x^{2p+1} - 1 = 0 \] を考える．このとき，\(x = 1\) は常に根の1つであるから，\(x - 1\) で割ると，次のような \(2p\) 次の方程式が得られる： \[ x^{2p} + x^{2p-1} + x^{2p-2} + \cdots + x^{2} + x + 1 = 0. \] この方程式は，\(x^{p}\) で割って，中央の項から等距離にある項を対にすることで，次数 \(p\) の方程式に帰着できる．すなわち： \[ x^{p} + \frac{1}{x^{p}} + x^{p-1} + \frac{1}{x^{p-1}} + \cdots + x^2+\frac{1}{x^2}+ x + \frac{1}{x} + 1 = 0. \] ここで \(x + \frac{1}{x} = y\) とおけば，\(y\) の冪乗を展開することで次のような関係が得られる： \[y^{2} = x^{2} + \frac{1}{x^{2}} + 2,\quad y^{3} = x^{3} + \frac{1}{x^{3}} + 3\left(x + \frac{1}{x}\right), \dots, \] すなわち， \[x^{2} + \frac{1}{x^{2}} = y^{2} - 2,\quad x^{3} + \frac{1}{x^{3}} = y^{3} - 3y,\] そして一般に， \[ x^{r} + \frac{1}{x^{r}} = y^{r} - r y^{r-2} + \frac{r(r - 3)}{2} y^{r-4} - \frac{r(r - 4)(r - 5)}{2 \cdot 3} y^{r-6} + \cdots \] が得られる．この展開は，\(y\) の指数が正である限り続く．[*2]

このようにして得られた式を上記の方程式に代入すれば，\(y\) についての \(p\) 次の方程式が得られる．この方程式を解けば \(y\) の値が \(p\) 個得られ，それぞれに対して \[ x^{2} - x y + 1 = 0 \] という2次方程式を解くことで \(x\) の値が2つずつ得られる．したがって，\(2p\) 個の \(x\) の値が得られ，これに \(x = 1\) を加えれば， \[x^{2p+1} - 1 = 0\] のすべての根が得られることになる．

こうして，平方根の開平操作のみによって次の方程式の根が得られる： \[ x^{2} - 1 = 0, \quad x^{3} - 1 = 0, \quad x^{5} - 1 = 0. \] したがって，\(n\) が \(2, 3, 5\) 以外の素因数を含まないとき，すなわち \(n = 2^{\lambda} \cdot 3^{\mu} \cdot 5^{\nu}\) の形であるときには， \[x^{n} - 1 = 0\] を解くことができる．

さらに，3次方程式の解法を援用すれば， \[x^{7} - 1 = 0\] も解けるので，\(n = 2^{\lambda} \cdot 3^{\mu} \cdot 5^{\nu} \cdot 7^{\varpi}\) の形のときも \[x^{n} - 1 = 0\] は解けることになる．

22．方程式 \[ x^{2p} + x^{2p-1} + \cdots + x + 1 = 0 \] の次数を \(p\) 次に還元するために我々が用いた方法は，一般に，次数が偶数であり，中央の項から等距離にある項の係数が等しいような方程式すべてに適用できる． なぜなら，方程式 \[ x^{2p} + a x^{2p-1} + b x^{2p-2} + \cdots + b x^{2} + a x + 1 = 0 \] を \(x^{p}\) で割ると，次の形に書き換えられる： \[ x^{p} + \frac{1}{x^{p}} + a\left(x^{p-1} + \frac{1}{x^{p-1}}\right) + b\left(x^{p-2} + \frac{1}{x^{p-2}}\right) + \cdots = 0. \] したがって，ここでも \[ x + \frac{1}{x} = y, \quad x^{2} + \frac{1}{x^{2}} = y^{2} - 2, \dots \] といった代入を用いることができ，この変換によって得られる \(y\) についての方程式は次数 \(p\) である．

また，次数が奇数 \(2p + 1\) の方程式 \[ x^{2p+1} + a x^{2p} + b x^{2p-1} + \cdots + h x^{p+1} + h x^{p} + \cdots + b x^{2} + a x + 1 = 0 \] を考えると，まず次のように整理できる： \[ x^{2p+1} + 1 + a x(x^{2p-1} + 1) + b x^{2}(x^{2p-3} + 1) + \cdots + h x^{p}(x + 1) = 0. \] ここで各項は \(x + 1\) で割り切れることがわかるので，これで割ると次のような式が得られる： \[\left. \begin{aligned} x^{2p} &+ x^{2p-1} + x^{2p-2} + x^{2p-3} + \cdots + 1 \\ &+ a x\,(x^{2p-2} + x^{2p-3} + \cdots + 1) \\ &+ b x^{2}(x^{2p-4} + x^{2p-5} + \cdots + 1) \\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots \\ &+ h x^{p} \end{aligned}\right\}=0.\,\,^{\text{[*1]}} \] この式を \(x\) の冪について整理すれば，先述した〔偶数次の〕方程式と同様の形が得られ，同様の手続きによって次数を \(p\)次 に還元することができる．

このような性質を最初に指摘したのは，おそらくド・モアブル氏であろう．彼は著書『解析雑録（Miscellanea analytica）』において，次数をもとの方程式の半分に変換する一般公式を提示している．この種の方程式がなぜこのような簡約を許容するのか．そのアプリオリな理由については，後に詳述することにしよう．

  1. 以下のように符号の一部はマイナスが正しい． \[{\displaystyle\left. \begin{aligned} x^{2p}&-x^{2p-1}+x^{2p-2}-x^{2p-3}+\dots+1\\ &+ax\,\,(x^{2p-2}-x^{2p-3}+\dots+1)\\ &+bx^2(x^{2p-4}-x^{2p-5}+\dots+1)\\ &\cdots\cdots\cdots\cdots\cdots\cdots\cdots\cdots\cdots\cdots\\ &+hx^p \end{aligned}\right\}=0.}\]

23．第21項で得られた式 \[ x^{r} + \frac{1}{x^{r}} = y^{r} - r y^{r-2} + \frac{r(r - 3)}{2} y^{r-4} - \dots \] を再び取り上げよう． よく知られた多倍角の定理によれば，右辺の量 \[ y^{r} - r y^{r-2} + \frac{r(r - 3)}{2} y^{r-4} - \dots \] は，半径1の円において，\(y\) がある弧の補弦（\(180^\circ\)からの補角に対応する弦）であるとき，その\(r\)倍の弧の補弦を表している． この弧を \(2\varphi\) と名づければ， \[ y = 2\cos \varphi, \quad y^{r} - r y^{r-2} + \frac{r(r - 3)}{2} y^{r-4} - \dots = 2\cos r\varphi \] となる．補弦は，よく知られているように，角の半分の余弦の2倍に等しい．

したがって，\(x + \frac{1}{x} = 2\cos \varphi\) とすれば，一般に \(x^{n} + \frac{1}{x^{n}} = 2\cos n\varphi\) が成り立つ． このことから，次の2つの方程式 \begin{align*} &x^{2} - 2x\cos \varphi + 1 = 0,\\ &x^{2n} - 2x^{n}\cos n\varphi + 1 = 0 \end{align*} は同時に成立することになり，任意の \(n\) に対して，前者は後者の因数であることがわかる．これがコーツ氏の有名な定理の基礎である．

さて，これら2つの方程式を2次方程式の解法に従って解けば， \begin{align*} &x = \cos \varphi \pm \sin \varphi \sqrt{-1}, \\ & x^{n} = \cos n\varphi \pm \sin n\varphi \sqrt{-1} \end{align*} が得られる．ここで，両方の式において符号が一致する必要があることは容易に示せる． 実際，\(\varphi\) を非常に小さいと仮定し，2次以上の微小量を無視すれば， \[ \cos \varphi = 1, \quad \sin \varphi = \varphi, \quad\text{同様に}\quad \cos n\varphi = 1, \quad \sin n\varphi = n\varphi \] となるので，2つの式は \begin{align*} &x = 1 \pm \varphi \sqrt{-1}, \\ & x^{n} = 1 \pm n\varphi \sqrt{-1} \end{align*} となる． ここで，前者を \(n\) 乗し，\(\varphi\) の2乗以上の項を無視すれば， \[ x^{n} = 1 \pm n\varphi \sqrt{-1} \] が得られ，これら2つの式における複号の正負（\(+\) および \(-\)）は一致していなければならないと結論される． したがって，符号の多義性を除外して考えると， \[ x = \cos \varphi + \sin \varphi \sqrt{-1} \] は，方程式 \[ x^{n} - \cos n\varphi - \sin n\varphi \sqrt{-1} = 0 \] の根であることが明らかである．

ここで，\(\sin n\varphi = 0\), \(\cos n\varphi = 1\) とすれば，\(n\varphi = 360^{\circ}\) または \(720^{\circ}\)，あるいは一般に \(n\varphi = m \times 360^{\circ}\)（\(m\) は任意の整数）となる．このとき，方程式 \[x^{n} - 1 = 0\] が得られ，その根は\(\varphi = \frac{m}{n} \times 360^{\circ}\)により，次のように表される： \[ x = \cos\left(\frac{m}{n} \times 360^{\circ}\right) + \sin\left(\frac{m}{n} \times 360^{\circ}\right)\sqrt{-1}. \] これは，方程式 \(x^{n} - 1 = 0\) の各根についての一般式であり，\(m = 1, 2, 3, \dots, n\) と順にとればすべての根が得られる．\(m > n\) としても，\(m < n\,\)[*1]のときと同じ値が再び現れるだけなので，それ以上の値をとる必要はない．

  1. 正確には \(m\leqq n\)．

24．この解についてまず指摘しておくべきは，方程式\(x^{n} - 1 = 0\)の根はすべて相異なるということである．なぜなら，単位円周上には，異なる弧であって同時に正弦と余弦の値が等しいものは存在しないからである． さらに明らかなように，\(m = n\) に対応する最後の根（\(x = 1\)）および，\(n\) が偶数のときの \(m = \frac{n}{2}\) に対応する根（\(x = -1\)）を除いて，すべての根は虚数である．実際，\(x\) の表現において虚部が消えるためには， \[ \sin\left(\frac{m}{n} \times 360^{\circ}\right) = 0 \] でなければならず，これは弧が \(360^{\circ}\) または \(180^{\circ}\) のときに限られる．これは， \(\frac{m}{n} = 1\)または\(=\frac{1}{2}\)， すなわち \(m = n\) または \(m = \frac{n}{2}\) のときである．前者では実部の \( \cos\left(\frac{m}{n} \times 360^{\circ}\right)\)は\(\cos 360^{\circ} = 1 \) になり，後者では \( \cos 180^{\circ} = -1 \) となる．

ここで \[ \alpha = \cos \frac{360^{\circ}}{n} + \sin \frac{360^{\circ}}{n}\,\sqrt{-1} \] とおけば，先の式により \[ \alpha^{m} = \cos\left(\frac{m}{n} \times 360^{\circ}\right) + \sin\left(\frac{m}{n} \times 360^{\circ}\right)\sqrt{-1} \] となるので，方程式 \(x^{n} - 1 = 0\) の異なる根はすべて \(\alpha\) の冪で表される．すなわち， \( \alpha, \alpha^{2}, \alpha^{3}, \dots, \alpha^{n} \) であり，最後の \(\alpha^{n}\) は常に \(1\) に等しい．このことは，まさに方程式 \(x^n - 1 = 0\) 自体から明らかであり，その方程式からは当然 \(\alpha^n - 1 = 0\) が得られる． また，\(n\) が偶数であるときに \(\alpha^{\frac{n}{2}}\) で表される根は，先述のとおり \(-1\) に等しい．

ここで注目すべきは，\(n\) が素数である場合，方程式 \(x^{n} - 1 = 0\) のすべての根は，そのいずれかの根（ただし最後の根は除く）の累乗によって常に表現することができるということである． たとえば \(n = 3\) の場合，根は \(\alpha, \alpha^{2}, \alpha^{3}\) となる．\(\alpha\) の代わりに次の根 \(\alpha^{2}\) をとれば，得られる根は \(\alpha^{2}, \alpha^{4}, \alpha^{6}\) となる．だが \(\alpha^{3} = 1\) であることから，\(\alpha^{4} = \alpha\)，また \(\alpha^{6} = \alpha^{3}\) であるのは明らかである．したがって，これらの根は再び \(\alpha^{2}, \alpha, \alpha^{3}\) となり，先ほどと同じである． 同様に，\(n = 5\) の場合，根は \(\alpha, \alpha^{2}, \alpha^{3}, \alpha^{4}, \alpha^{5}\) である．\(\alpha\) の代わりに \(\alpha^{2}\) をとれば，得られるのは \(\alpha^{2}, \alpha^{4}, \alpha^{6}, \alpha^{8}, \alpha^{10}\) であるが，\(\alpha^{5} = 1\) であるため，これらはそれぞれ \(\alpha^{2}, \alpha^{4}, \alpha, \alpha^{3}, \alpha^{5}\) に一致する． さらに，\(\alpha\) の代わりに \(\alpha^{3}\) をとった場合も同様に，\(\alpha\) の指数が \(5\) を超えるたびに可能な限り \(5\) を差し引けば，得られる根は \(\alpha^{3}, \alpha, \alpha^{4}, \alpha^{2}, \alpha^{5}\) である． 最後に，\(\alpha\) の代わりに \(\alpha^{4}\) をとると，\(\alpha^{4}, \alpha^{3}, \alpha^{2}, \alpha, \alpha^{5}\) となり，結局どの場合でも同じ根が異なる順序で現れることになる．

一般に，\(\alpha^{m}\) を \(\alpha, \alpha^{2}, \alpha^{3}, \dots, \alpha^{n}\) のうちの任意の1つとし，\(m < n\) かつ \(n\) が素数であるとする．この \(\alpha^{m}\) を \(\alpha\) の代わりに用いれば，得られる根は \(\alpha^{m},\ \alpha^{2m},\ \alpha^{3m},\ \dots,\ \alpha^{nm}\) となる．ここで，\(2m, 3m, 4m, \dots\) のような指数が \(n\) を超えるとき，それらから \(n\) の最大の倍数を引いた余りを \(p, q, r, \dots\) と置けば，得られる根は \(\alpha^{m},\ \alpha^{p},\ \alpha^{q},\ \alpha^{r},\ \dots,\ \alpha^{n}\) となる．そして，私は，\(m, p, q, r, \dots, n\) はすべて \(n\) 以下の整数であり，かつ互いに異なることを主張する． 実際，仮に \(p\) と \(r\) のような2つの数が等しいとすれば，それらは \(2m, 4m\) のような数から \(n\) の倍数を引いた余りであるから，\(2m\) と \(4m\) の差が \(n\) で割り切れることになる．しかし，\(n\) が素数で \(m < n\) である以上，これはあり得ない． したがって，\(m, p, q, r, \dots\)[*1]はすべて異なり，かつ \(n\) 未満の \(n - 1\) 個の整数であるから，それらは \(1, 2, 3, \dots, n - 1\) に他ならない．よって，\(\alpha^{m}, \alpha^{p}, \alpha^{q}, \alpha^{r}, \dots, \alpha^{n}\) は \(\alpha, \alpha^{2}, \alpha^{3}, \dots, \alpha^{n}\) と同じ根を与えることになる． この証明は，\(n\) が素数でない場合にも，\(m\) と \(n\) と互いに素である限り，常に成り立つ．しかし，\(m\) が \(n\) と互いに素でないとき，すなわち最大公約数が \(l\) のときは，\(m, p, q, r, \dots\) はすべて \(l\) で割り切れるので，それらは \(n\) 未満の \(l\) の倍数に限られる．

このことから，一般に，\(m\)と\(n\)が互いに素である限り，\(x^{n} - 1 = 0\) のすべての根 \(\alpha, \alpha^{2}, \alpha^{3}, \dots, \alpha^{n}\) は，\(\alpha^{m}\) の \(1\) 次，\(2\) 次，\(3\) 次，…，\(n\) 次の冪として表すことができることが容易に結論できる． 一方，\(m\) が \(n\) と互いに素でないとき，最大公約数を \(l\) とすれば，得られる根は \(\alpha^{l},\ \alpha^{2l},\ \alpha^{3l},\ \dots,\ \alpha^{n}\) に限られ，それぞれは \(\frac{n}{l}\) 回ずつ現れる．また，これらの根は \(lf = n\) とすれば，方程式 \(x^{f} - 1 = 0\) の根でもあることが，先の式から明らかである．

さらに，\(x^{n} - 1 = 0\) の根は \[\alpha,\,\, \alpha^{2},\,\, \alpha^{3},\,\, \dots,\,\, \alpha^{n}\] であり，\(\alpha^{n} = 1\) であるから，それらは \[ \frac{1}{\alpha},\quad \frac{1}{\alpha^{2}},\quad \frac{1}{\alpha^{3}}, \dots,\quad \frac{1}{\alpha^{n}} \] としても表せる．実際， \(\frac{1}{\alpha} = \alpha^{n - 1},\, \frac{1}{\alpha^{2}} = \alpha^{n - 2}, \dots\) である．

また，\(x^{n} - 1 = 0\) は第2項を欠く方程式であるため，一般に \[ \alpha + \alpha^{2} + \alpha^{3} + \cdots + \alpha^{n} = 0 \] が成り立ち，同様に \[ \frac{1}{\alpha} + \frac{1}{\alpha^{2}} + \frac{1}{\alpha^{3}} + \cdots + \frac{1}{\alpha^{n}} = 0 \] も成り立つ． そして，\(n = lf\) のように合成数であるときには，特に \[ \begin{aligned} &\alpha^{l} + \alpha^{2l} + \alpha^{3l} + \cdots + \alpha^{n} = 0, \\[+3pt] &\frac{1}{\alpha^{f}} + \frac{1}{\alpha^{2f}} + \frac{1}{\alpha^{3f}} + \cdots + \frac{1}{\alpha^{n}} = 0,\,\,^{\text{[*2]}} \end{aligned} \] 同様に， \[ \begin{aligned} &\alpha^{f} + \alpha^{2f} + \alpha^{3f} + \cdots + \alpha^{n} = 0, \\[+3pt] &\frac{1}{\alpha^{f}} + \frac{1}{\alpha^{2f}} + \frac{1}{\alpha^{3f}} + \cdots + \frac{1}{\alpha^{n}} = 0\qquad \end{aligned} \]が成り立つ． すなわち，指数が \(l\) または \(f\) の倍数であるような冪（正でも負でも）の和は 0 に等しくなる．したがって，\(l\) で割り切れない指数の冪の和もまた 0 であり，\(f\) で割り切れないものについても同様である． これらのさまざまな注意は，今後の議論において有用となるであろう．

  1. ここに\(n\)は含めていないことに注意．
  2. \(f\)ではなく\(l\)であろう．\(\dfrac{1}{\alpha^{l}} + \dfrac{1}{\alpha^{2l}} + \dfrac{1}{\alpha^{3l}} + \dots + \dfrac{1}{\alpha^{n}} = 0.\)

25．ここに，方程式 \(x^{n} - 1 = 0\) に対する \(\alpha\) の値を，\(n = 1\)[*1]から \(n = 6\) まで示す： \[ \begin{aligned} n = 2 \quad & \left\{ \begin{aligned} &\alpha = -1, \\ &\alpha^{2} = 1; \end{aligned} \right. \\ n = 3 \quad & \left\{ \begin{aligned} &\alpha = \frac{-1 + \sqrt{-3}}{2}, \\ &\alpha^{2} = \frac{-1 - \sqrt{-3}}{2}, \\ &\alpha^{3} = 1; \end{aligned} \right. \\ n = 4 \quad & \left\{ \begin{aligned} &\alpha = \sqrt{-1}, \\ &\alpha^{2} = -1, \\ &\alpha^{3} = -\sqrt{-1}, \\ &\alpha^{4} = 1; \end{aligned} \right. \\ n = 5 \quad & \left\{ \begin{aligned} &\alpha = \frac{\sqrt{5} - 1}{4} + \frac{\sqrt{10 + 2\sqrt{5}}}{4} \sqrt{-1}, \\ &\alpha^{2} = -\frac{\sqrt{5} + 1}{4} + \frac{\sqrt{10 - 2\sqrt{5}}}{4} \sqrt{-1}, \\ &\alpha^{3} = -\frac{\sqrt{5} + 1}{4} - \frac{\sqrt{10 - 2\sqrt{5}}}{4} \sqrt{-1}, \\ &\alpha^{4} = \frac{\sqrt{5} - 1}{4} - \frac{\sqrt{10 + 2\sqrt{5}}}{4} \sqrt{-1}, \\ &\alpha^{5} = 1; \end{aligned} \right.\\ n = 6 \quad & \left\{ \begin{aligned} &\alpha = \frac{1 + \sqrt{-3}}{2}, \\ &\alpha^{2} = \frac{-1 + \sqrt{-3}}{2}, \\ &\alpha^{3} = -1, \\ &\alpha^{4} = \frac{-1 - \sqrt{-3}}{2}, \\ &\alpha^{5} = \frac{1 - \sqrt{-3}}{2}, \\ &\alpha^{6} = 1. \end{aligned} \right. \end{aligned} \] もし \(x^{7} - 1 = 0\) に対する \(\alpha\) の値を求めたいならば，先に述べたように，3次方程式の解法を要することになる．実際，第21項の公式において \(p = 3\) とすれば，\(y\) についての変換方程式は \[ y^{3} + y^{2} - 2y - 1 = 0 \] となる．これは3次方程式であり，必ず実根を持つ．その実根を方程式 \[ x^{2} - x y + 1 = 0 \] に代入することで \(x\) を求めることができ， \[ \alpha = \frac{y + \sqrt{y^{2} - 4}}{2} \] を得る． また，方程式 \[x^{8} - 1 = 0, \quad x^{9} - 1 = 0,\quad x^{10} - 1 = 0\] に対しても，\(\alpha\) は平方根のみ[*2]を用いて表すことができる． しかし，\(x^{11} - 1 = 0\) の場合には，5次方程式 \[ y^{5} + y^{4} - 4y^{3} - 3y^{2} + 3y + 1 = 0 \] を解く必要がある．\(y\) が得られたと仮定すれば，\(\alpha\) は上と同様の式で表されることになる．

  1. \(n=2\) であろう．
  2. \(x^9-1=0\) には3乗根も必要であろう．
