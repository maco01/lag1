65．さて，第51項の公式に立ち返ろう． 与えられた方程式 (\(a\)) は，方程式 (\(c\))と(\(d\)) から \(y\) を消去した結果として捉えることができるのは明らかである． すなわち，\(y\) についての方程式の係数 \({A}, {B}, {C}, \dots\) を与えられたものとし， \(x\) を \(y\) の関数として表す式の係数 \({F}, {G}, \dots\) を未定係数と見なせば， \(y\) を消去して得られる方程式の項と，与えられた方程式の項とを比較することによって， これらの未定係数を決定することができる． ただし，その未定係数の数が \(\mu\) 以上でなくてはならないが， この条件は，\(\lambda = \frac{\mu}{2}\) または \(\frac{\mu - 1}{2}\) とすれば常に満たされるので， 心配する必要はない． そして，\(y\) についての方程式が解ける形で与えられていれば（これは無数の方法で可能である）， 与えられた方程式の完全な解法が得られることになる． ただし，困難は未定係数 \({F}, {G}, \dots\) の決定にある．

しかしながら，方程式（\(d\)）によって与えられる \(x\) の式を，\(y\) が分子のみに現れる形に書き換えれば，係数の決定と \(y\) の消去はかなり容易になる．このような変形は，\(y\) の適切な多項式を分数 \[ \frac{F + G y + H y^{2} + \cdots + K y^{\lambda}}{L + M y + N y^{2} + \cdots + R y^{\lambda}} \] の分子と分母に掛けることで常に可能であることが，以下のようにしてわかる．

次のように置く． \[ z = L + M y + N y^{2} + \dots + R y^{\lambda}. \] \(y\) は方程式 \[ y^{\mu} + A y^{\mu - 1} + B y^{\mu - 2} + \dots = 0 \] によって定まるので，これを用いて \(y\) を消去することで，\(z\) に関する次数 \(\mu\) の方程式が得られる．それを次のように表す： \[ z^{\mu} + \alpha z^{\mu - 1} + \beta z^{\mu - 2} + \gamma z^{\mu - 3} + \dots - \omega = 0. \] ここで係数 \( \alpha，\beta，\gamma，\ldots，\omega \) は，係数 \( A，B，C，\ldots \) および \( L，M，N，\ldots \) の既知の関数となる．

したがって，次のように書くことができる： \[ z \left( z^{\mu - 1} + \alpha z^{\mu - 2} + \beta z^{\mu - 3} + \cdots \right) = \omega. \] これより，\(z\) に多項式 \[z^{\mu - 1} + \alpha z^{\mu - 2} + \beta z^{\mu - 3} + \cdots\] を掛けると \(\omega\)（すなわち，\(y\) に依存しない定数）に等しくなることがわかる． したがって，この多項式において \(z\) をその \(y\) による式に戻せば， 求めるべき多項式が得られる． この多項式は，必要であれば \(y^{\mu}\) より低い冪のみを含むようにできる． というのも，方程式 \[ y^{\mu} + A y^{\mu - 1} + \dots = 0 \] を用いれば，\(y^{\mu}\)以上の次数の冪は常に より低い次数の冪に還元できるからである．

このようにして，方程式 (\(d\)) は次の形に帰着される： \[ x = a + b y + c y^2 + \dots + k y^{\mu - 1}. \tag{\(k\)} \] したがって，与えられた方程式 (\(a\)) \[ x^{\mu} + m x^{\mu - 1} + n x^{\mu - 2} + \dots = 0 \] は，常に，方程式 (\(c\)) \[ y^{\mu} + A y^{\mu - 1} + B y^{\mu - 2} + \dots + V = 0 \] および方程式 (\(k\)) を用いて \(y\) を消去した結果として捉えることができる． このとき，係数 \(a, b, c, \dots, k\)（その数は \(\mu\) 個）を未定とすれば， \(y\) を消去して得られる方程式と与えられた方程式を項ごとに比較することで， \(\mu\) 個の条件が得られ，それによって \(a, b, c, \dots\) を決定することができる．

もし \(y\) についての方程式を \[ y^{\mu} + V = 0 \] のように2項だけにすれば，この方法はオイラー氏およびベズー氏の方法に帰着する． この方法については，すでに本論文の中で何度か言及してきた．

ここで述べた詳細は，この方法をチルンハウス氏の方法に結びつけ， 両者の類似性と相互の依存関係を示すものである．

66．すべては，与えられた方程式の項と\(y\) を消去して得られる方程式の項とを比較することにより， \(\mu\)個の未知数 \(a, b, c, \dots, k\)を決定することに帰着する． まず，この消去後の方程式について，以下の点に注目しよう． この方程式は，\(a, b, c, \dots, k\) および \(x\) に関する多項式として表され， 第13項で述べた消去理論から容易に導かれるように， どの項においてもこれらの文字についての次数は常に\(\mu\)になる．

したがって，この方程式を \(x\) について整理すれば， 各項の係数は \(a, b, c, \dots, k\) に関する同次の多項式となり， その次数は \(x^{\mu}, x^{\mu - 1}, x^{\mu - 2}, \dots\) に対して それぞれ \(0, 1, 2, 3, \dots\) となる．

すなわち，第1項 \(x^{\mu}\) の係数は 1 であり， 第2項 \(x^{\mu - 1}\) の係数は，\(\alpha, \beta, \gamma, \dots\) を数値係数として， \[ \alpha a + \beta b + \gamma c + \dots \] の形であり， 第3項 \(x^{\mu - 2}\) の係数は \[ \alpha a^2 + \beta a b + \gamma b^2 + \delta a c + \dots \] の形であり，…等々以下同様である．

したがって，第2項の係数を \(m\) に，第3項の係数を \(n\) に，…と順に等置すれば， \(\mu\) 個の未知数 \(a, b, c, \dots, k\) についての \(\mu\) 個の方程式が得られる． このうち，第1の方程式は1次式，第2は2次式，第3は3次式，…等々となるので， これらの未知数のうち任意の1つを残して他を消去すれば， その最終方程式の次数は，一般に \(1 \cdot 2 \cdot 3 \cdots \mu\) となる． これはオイラー氏の見解には反するが，ベズー氏が帰納的に見出した結果とは一致している．

67．係数 \(a, b, c, \dots, k\) についての方程式の次数に関する結論をさらに裏づけ， またそれらの方程式が簡約可能となる場合を見極めるために， これらの量 \(a, b, c, \dots\) を与えられた方程式の根 \(x', x'', x''', \dots\) で表す式を アプリオリに求めてみよう．

第54項と同様に，\(u = \sqrt[\mu]{-V}\)と置く． また， \[y^{\mu} - 1 = 0\] の \(\mu\) 個の根を \(1, \alpha, \beta, \gamma, \dots\) とすれば， \[y^{\mu} + V = 0\] の \(\mu\) 個の根は \(u, \alpha u, \beta u, \gamma u, \dots\) となる． これらの根を順に方程式 (\(k\)) の \(y\) に代入し， 同時に \(x', x'', x''', \dots\) を \(x\) に代入すれば，次の \(\mu\) 個の方程式が得られる： \begin{align*} x'\,\,&=a+bu+cu^2+du^3+\dots+ku^{\mu-1},\\ x''\,&=a+\alpha bu+\alpha^2cu^2+\alpha^3du^3+\dots+\alpha^{\mu-1}ku^{\mu-1},\\ x'''&=a+\beta bu+\beta^2cu^2+\beta^3du^3+\dots+\beta^{\mu-1}ku^{\mu-1},\\ x^{\mathrm {IV}}&=a+\gamma bu+\,\gamma^2cu^2+\gamma^3du^3+\dots+\gamma^{\mu-1}ku^{\mu-1},\\ \dots&\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots \end{align*} この連立方程式によって，未知数 \(a, b, c, \dots\) を決定することができる．

この決定には何の困難もない．というのも，\(1, \alpha, \beta, \gamma, \dots\) は 方程式 \(y^{\mu} - 1 = 0\) の根であり，この方程式は中間項をすべて欠いているので， よく知られているように次の関係が成り立つ： \begin{align*} &1 + \,\alpha \,+ \,\,\beta \,+\,\gamma \,\,+ \dots = 0,\\ &1 + \alpha^2 + \beta^2 +\gamma^2 + \dots = 0,\\ &1 + \alpha^3 + \beta^3 +\gamma^3 + \dots = 0,\\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots \end{align*} すなわち，すべての根をある同じ冪に上げたものの和は， その指数が \(\mu\) で割り切れない限り常に 0 になる． 一方，指数が \(\mu\) の倍数である場合には， 方程式 \(y^{\mu} - 1 = 0\) から明らかなように， \(\alpha^{\mu} = 1, \alpha^{2\mu} = 1, \dots\) 等々が成り立つ．

したがって，前項[*1]の \(\mu\) 個の方程式に，それぞれ対応する \(\mu\) 個の根 \(1, \alpha, \beta, \dots\) を \(\mu\) 乗，\((\mu - 1)\) 乗，\((\mu - 2)\) 乗，…，1 乗したものを順に掛けて 加え合わせることで，次のような式が得られる： \begin{align*} &\mu \,a\,\,\,\,\,\,=x'+x''+x'''+x^{\mathrm {IV}}+\dots,\\ &\mu \,ub\,\,\,=x'+\alpha^{\mu-1}x''+\beta^{\mu-1}x'''+\gamma^{\mu-1}x^{\mathrm {IV}}+\dots,\\ &\mu \,u^2c=x'+\alpha^{\mu-2}x''+\beta^{\mu-2}x'''+\gamma^{\mu-2}x^{\mathrm {IV}}+\dots,\\ &\mu \,u^3d=x'+\alpha^{\mu-3}x''+\beta^{\mu-3}x'''+\gamma^{\mu-3}x^{\mathrm {IV}}+\dots,\\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots \end{align*}

まず，\(a\) は 根\(x', x'', \dots\) をどのように置換しても値が変わらないので， 1次方程式[*2]によって与えられることがわかる． 実際， \[ x' + x'' + x''' + \dots = -m \] であるから， \(a = -\frac{m}{\mu}\) となる．

一方，\(u b, u^2 c, u^3 d, \dots\) の各量は，一般には， \(\mu\) 個の根\(x', x'', x''', \dots\) のあらゆる可能な置換（順列）の数――よく知られているように，\(1 \cdot 2 \cdot 3 \cdots \mu\)である――と同じ次数の方程式に従う．なぜなら，それぞれの置換に対して \(u b, u^2 c, u^3 d, \dots\)は異なる値をとるからである． ただし，これらの値の間に何らかの関係があれば， それらを根とする方程式の次数を下げることが可能になる． これについて，次に検討することにしよう．

69．一般に次のように置こう： \[ t = x' + \alpha x'' + \alpha^2 x''' + \alpha^3 x^{\mathrm{IV}} + \dots + \alpha^{\mu - 1} x^{(\mu)}. \] そして，\(t\) についての方程式の性質がどのようなものであるべきかを考察する． そのために，\(\mu\) 個の根 \(x', x'', \dots\) がとりうる \(1 \cdot 2 \cdot 3 \cdots \mu\) 通りの置換（順列）によって \(t\) がとりうる個々の値をすべて求める． この考察では，第55項と同様の方法を用いる． まず \(x'\) を固定し，残りの \(\mu - 1\) 個の変数の位置を変える． これらは \(1 \cdot 2 \cdot 3 \cdots (\mu - 1)\) 通りの置換（順列）をもつので， \(t\) の特定の値 \(t', t'', t''', \dots\) がそれぞれ得られる． 次に，これらの各式において \(x'\) の位置を順に \(x'', x''', \dots\) の位置に移動させることで， 求めるべき \(1 \cdot 2 \cdot 3 \cdots \mu\) 個のすべての値が得られ， それらが \(t\) についての方程式の根となる．

ところが，これらすべての値を網羅するにはは，\(t', t'', t''', \dots\) のそれぞれに \(\alpha, \alpha^2, \alpha^3, \dots, \alpha^{\mu - 1}\) を順次掛けていけばよいことが容易にわかる． したがって，\(t\) についての方程式の根は次のように表される： \[ \begin{array}{lllll} t', & \alpha t', & \alpha^2 t', & \alpha^3 t', \dots, & \alpha^{\mu - 1} t', \\ t'', & \alpha t'', & \alpha^2 t'', & \alpha^3 t'', \dots, & \alpha^{\mu - 1} t'', \\ t''', & \alpha t''', & \alpha^2 t''', & \alpha^3 t''', \dots, & \alpha^{\mu - 1} t''', \\ \dots&\dots&\dots&\,\,\dots\,\,\,\,\,\dots&\dots \end{array} \] ここから，\(t\) についての方程式には，\(t\) の冪乗のうち，指数が \(\mu\) の倍数のものしか含まれないことが容易に結論づけられる．

したがって，\(t^{\mu} = \theta\) とおけば， \[ \theta = \left( x' + \alpha x'' + \alpha^2 x''' + \alpha^3 x^{\mathrm{IV}} + \dots \right)^{\mu} \] となり，\(\theta\) についての方程式は \(1 \cdot 2 \cdot 3 \cdots (\mu - 1)\) 次となる． この方程式の根は，\(x'\) を除いた \(\mu - 1\) 個の根 \(x'', x''', \dots\) の置換によって 生じる \(\theta\) の諸値である．

この結論は，\(\mu\) がどのような数であっても成り立つ． 以下では，\(\mu\) が素数である場合とそうでない場合とを分けて考察することにする．

70．まず，指数 \(\mu\) が素数であるとしよう． このとき，\(\theta\) のすべての値を求めるには， \(\mu - 2\) 個の根\(x''', x^{\mathrm{IV}}, \dots\) の間の置換によって生じる値を調べれば十分である． これらの置換（順列）の数は \(1 \cdot 2 \cdot 3 \cdots (\mu - 2)\) 通りである． そして，それぞれの \(\theta\) の値の式において，\(\alpha\) を順次 \(\alpha^2, \alpha^3, \dots, \alpha^{\mu - 1}\) に置き換えていけばよい． このことは，第56項と同様の議論によって確認できる．

ところで， \[ \theta^0 = \xi + \xi' + \xi'' + \xi''' + \dots + \xi^{(\mu - 1)} = \left[x' + x'' + x''' + \dots + x^{(\mu - 1)}\,^{\text{[*1]}}\right]^{\mu} = (-m)^{\mu}. \] したがって，\(\mu \xi, \mu \xi_2, \mu \xi_3, \dots\) から それぞれ \((-m)^{\mu}, (-m)^{2\mu}, (-m)^{3\mu}, \dots\) を引いた差 \[ \mu \xi - (-m)^{\mu},\quad \mu \xi_2 - (-m)^{2\mu},\quad \mu \xi_3 - (-m)^{3\mu}, \dots \] は，\(\mu-1\) 個の根 \(\theta', \theta'', \theta''', \dots\) の和，およびそれらの2乗の和，3乗の和，…等々に等しい． したがって，よく知られた公式[*2]により次のようになる： \[ \begin{aligned} &T = \mu \xi - (-m)^{\mu}, \\ &U = \frac{T \left[\mu \xi - (-m)^{\mu} \right]}{2} - \frac{\mu \xi_2 - (-m)^{2\mu}}{2}, \\ &X = \frac{U \left[ \mu \xi - (-m)^{\mu} \right]}{3} - \frac{T \left[ \mu \xi_2 - (-m)^{2\mu} \right]}{3} + \frac{\mu \xi_3 - (-m)^{3\mu}}{3}, \\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots \end{aligned} \]

73．さて，\(T, U, X, \dots\) の各式において根 \(x', x'', x''', \dots\) のあらゆる可能な置換を考えても， それぞれの量の異なる値は \(1 \cdot 2 \cdot 3 \cdots (\mu - 2)\) 個しか得られない． これらの値は，\(\mu - 2\) 個の根\(x''', x^{\mathrm{IV}}, \dots\) の置換だけから生じるものである． したがって，その数だけ \(\theta\) についての方程式 \[ \theta^{\mu - 1} - T \theta^{\mu - 2} + U \theta^{\mu - 3} - \dots = 0 \] があり，これらの方程式をすべて掛け合わせれば，\(\theta\) についての 次数 \(1 \cdot 2 \cdot 3 \cdots (\mu - 1)\) の方程式が得られる． そして，その方程式のすべての係数は，与えられた方程式の係数 \(m, n, p, \dots\) の 有理式として定まる．

このようにして \(\theta\) についての方程式が得られたならば， それを上記のような \(\mu - 1\) 次方程式で割ることにより， \(T, U, X, \dots\) の間に \(\mu - 1\) 個の条件式が得られる． これらを用いれば，たとえば \(U, X, \dots\) を \(T\) の式として表すことができ， \(T\) についての最終方程式が得られるが，その次数は \(1 \cdot 2 \cdot 3 \cdots (\mu - 2)\) を超えることはない．

実際，\(T\) は \(1 \cdot 2 \cdot 3 \cdots (\mu - 2)\) 通りの異なる値しかとり得ないので，簡略のため \[ \nu = 1 \cdot 2 \cdot 3 \cdots (\mu - 2) \] とおき，\(T\)の値を \(T', T'', T''', \dots, T^{(\nu)}\) とすれば，\(T\) についての方程式は次の形となる： \[ T^{\nu} - \varpi T^{\nu - 1} + \rho T^{\nu - 2} - \sigma T^{\nu - 3} + \dots = 0. \] この方程式の根は \(T', T'', T''', \dots\) であり，もし望むなら， それらの値から係数 \(\varpi, \rho, \sigma, \dots\) を アプリオリに決定することも可能である．

このようにして，\(\theta\) についての次数 \(\nu (\mu - 1)\) の方程式に頼ることなく， \(T\) についての方程式を直接得ることができる． また，この\(\theta\)についての方程式とは独立に，\(T\) を用いて他の係数 \(U, X, \dots\) を 求めることもできる．この点については第IV節で改めて示すことにする．

ところで，方程式 \(y^{\varpi} - 1 = 0\) は中間項を欠いているため，次の式が成り立つ： \[ \begin{aligned} &1 + \omega + \varphi + \psi + \dots = 0, \\ &1 + \omega^2 + \varphi^2 + \psi^2 + \dots = 0, \\ &1 + \omega^3 + \varphi^3 + \psi^3 + \dots = 0, \\ &\dots \dots \dots \dots \dots \dots \dots \dots \dots \end{aligned} \] したがって，第67項で用いたのと同様の手法によって， \(a, b, c, \dots, k\)，\(a', b', c', \dots, k'\)，\(a'', b'', c'', \dots, k''\)，… の値を求めることができる． その結果は次の通りである． \[ \begin{aligned} &\varpi a = z' + z'' + z''' + \dots + z^{(\varpi)}, \\ &\varpi b = z' + \omega^{\varpi - 1} z'' + \varphi^{\varpi - 1} z''' + \dots, \\ &\varpi c = z' + \omega^{\varpi - 2} z'' + \varphi^{\varpi - 2} z''' + \dots, \\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots, \end{aligned} \] 続いて， \[ \begin{aligned} &\varpi a' = u' + u'' + u''' + \dots + u^{(\varpi)}, \\ &\varpi b' = u' + \omega^{\varpi - 1} u'' + \varphi^{\varpi - 1} u''' + \dots, \\ &\varpi c' = u' + \omega^{\varpi - 2} u'' + \varphi^{\varpi - 2} u''' + \dots, \\ &\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots\dots, \end{aligned} \] …等々以下同様である．

82．係数 \(a, b, c, \dots, k\) の値を検討しよう． まず明らかなのは，\(\varpi a\) の値がすべての根 \(x', x'', x''', \dots, x^{(\mu)}\) の和に等しくなることである． したがって，\(\varpi a = -m\) なので， \(a = -\frac{m}{\varpi}\)．

次に，\(\omega^2, \omega^3, \dots\) を \(\varphi, \psi, \dots\) の代わりに用いて， 方程式 \(y^{\varpi} - 1 = 0\) の根を \(1, \omega, \omega^2, \omega^3, \dots, \omega^{\varpi - 1}\)（第24項）と表すと， 次の式が得られる． \[ \varpi k = z' + \omega z'' + \omega^2 z''' + \omega^3 z^{\mathrm{IV}} + \dots + \omega^{\varpi - 1} z^{(\varpi)}. \] そして，\(\varpi h, \varpi g, \dots\) の値を得るには，この式において \(\omega\) を順次 \(\omega^2, \omega^3, \dots\) に置き換えればよい．

ところで，この \(\varpi k\) の式は，第76項の量 \(a^{(\nu)}\) あるいは \(t\) の式と同じである． したがって，\(\varpi h, \varpi g, \dots\) の式も第76項の量 \(a^{(2\nu)}, a^{(3\nu)}, \dots\) と同じになる． このことから，次の結論を導くのは容易である． 第80項の方程式 (\(l\)) における係数 \(a, b, c, d, \dots\) を \(\varpi\) 倍したものは， それぞれ第65項の方程式 (\(k\)) における係数 \(a, b, c, \dots\) において， \(1, \nu, 2\nu, 3\nu, \dots, (\mu - \nu)\) 番目の位置を占める各係数を \(\mu\) 倍したものにそれぞれ等しい．

それゆえ，上記の式に現れる係数 \(a, b, c, \dots\) に対して， 第76項以降で得られた結論をそのまま適用することができる．

83．同じ式 (\(l\)) に現れる他の係数 \(a', b', c', \dots, a'', b'', c'', \dots\) については， 第79項と同様の考察により，それらを先の係数[*1]，あるいはそのうちの任意の1つに依存させることも可能である． また，第81項の式に基づいて，それぞれの係数が直接依存する方程式の次数と形を アプリオリに決定することもできる．

この目的のためには，\(u', u'', u''', \dots\) の各量が，対応する \(z', z'', z''', \dots\) の各量と同様に，同じ根の関数であり，それらの根の間でどのような置換を施しても不変であるという性質を持つことに注意すれば十分である． \(v', v'', v''', \dots\) についても同様である． したがって，係数 \(b', c', d', \dots, b'', c'', d'', \dots\) に対しては， 係数 \(b, c, d, \dots\) に関して行ったのと同様の議論と結論[*2]を適用することができる．

しかし，係数 \(a', a'', \dots\) に関しては別途考察が必要である． 第77項と同様の推論によって，これらの各量が \(\frac{1 \cdot 2 \cdot 3 \cdots \mu}{(1 \cdot 2 \cdot 3 \cdots \nu)^{\varpi}}\) 個の異なる値しか持たないことを示した上で， さらに次のことに着目する． すなわち，これらの量は \(u', u'', u''', \dots, u^{(\varpi)}\) や \(v', v'', v''', \dots, v^{(\varpi)}\) などの間の置換によっていかなる変化も受けない．それゆえ，係数 \(a', a'', \dots\) の異なる値の個数を得るには，\(\frac{1 \cdot 2 \cdot 3 \cdots \mu}{(1 \cdot 2 \cdot 3 \cdots \nu)^{\varpi}}\)をさらに \(1 \cdot 2 \cdot 3 \cdots \varpi\) で割る必要がある． したがって，これらの係数はそれぞれ，次数 \[ \frac{1 \cdot 2 \cdot 3 \cdots \mu}{1 \cdot 2 \cdot 3 \cdots \varpi \, (1 \cdot 2 \cdot 3 \cdots \nu)^{\varpi}} \] の特定の方程式によって決定されることになる．

84．与えられた方程式の次数が偶数であると仮定し，\(\varpi = 2\) とすれば，\(\mu = 2\nu\) となる． この場合，方程式 \(y^{\varpi} - 1 = 0\) は次のようになる： \[ y^2 - 1 = 0. \] これより2つの根 \[ y = 1 \quad \text{および} \quad y = -1 \] が得られる． したがって，前述の方法に従えば，与えられた方程式 \[ x^{2\nu} + m x^{2\nu - 1} + n x^{2\nu - 2} + p x^{2\nu - 3} + \dots = 0 \] は，次の2式の積となる： \[ \begin{aligned} &x^{\nu} - (a + b) x^{\nu - 1} + (a' + b') x^{\nu - 2} - (a'' + b'') x^{\nu - 3} + \dots = 0, \\ &x^{\nu} - (a - b) x^{\nu - 1} + (a' - b') x^{\nu - 2} - (a'' - b'') x^{\nu - 3} + \dots = 0. \end{aligned} \] これより， \[ a = -\frac{m}{2} \quad \text{および} \quad b = \frac{z' - z''}{2} \] が得られる．ここで， \[ \begin{aligned} z' &= x' + x''' + x^{\mathrm{V}} + \dots + x^{(2\nu - 1)}, \\ z'' &= x'' + x^{\mathrm{IV}} + x^{\mathrm{VI}} + \dots + x^{(2\nu)} \end{aligned} \] である．また，\(b\) についての方程式の次数は， \[ \frac{1 \cdot 2 \cdot 3 \cdots 2\nu}{(1 \cdot 2 \cdot 3 \cdots \nu)^2} \quad \text{すなわち} \quad \frac{(\nu + 1)(\nu + 2)(\nu + 3) \cdots 2\nu}{1 \cdot 2 \cdot 3 \cdots \nu} \] となり，すべての冪指数は偶数となる．

ゆえに，もし与えられた方程式が次のような \(\nu\)次 の因数をもつと仮定すれば： \[ x^{\nu} + m' x^{\nu - 1} + n' x^{\nu - 2} + p' x^{\nu - 3} + \dots = 0, \] 係数 \(m'\) についての方程式の次数は次のようになる： \[ \frac{(\nu + 1)(\nu + 2)(\nu + 3) \cdots (2\nu)}{1 \cdot 2 \cdot 3 \cdots \nu}. \] この数は，\(2\nu\) 個の対象から \(\nu\) 個を選ぶ組合せの数を表していることでも，つじつまが合っている．

また， \[-m' = a + b = -\frac{m}{2} + b\] とすれば， \(b\) についての方程式は偶数冪のみを含むので， \(m'\) についての方程式は，その第2項を消去すれば， 1つおきの項が同時に消えるような性質を持つことになる． これは，4次方程式に関して第35項で見た性質と同様である．

85．与えられた方程式が6次であるとし，\(\nu = 3\) とすれば， \(4b^2 = \theta\) と置くことで，\(\theta\) についての方程式の次数は， \[ \frac{4 \cdot 5 \cdot 6}{2 \times 1 \cdot 2 \cdot 3} = 10 \] となる．

ベズー氏は，この方程式が2次方程式を用いて2つの方程式に分解できると考えているが， 私はそれには強く疑問を抱く． 実際，\(\theta\) についての方程式の根は以下の10個の量によって表されるが， これらは， 6つの根 \(x', x'', x''', \dots\) の置換から生じる \((z' - z'')^2\) のすべての値である： \[ \begin{aligned} &(x' + x'' + x''' - x^{\mathrm{IV}} - x^{\mathrm{V}} - x^{\mathrm{VI}})^2, \\ &(x' + x'' + x^{\mathrm{IV}} - x''' - x^{\mathrm{V}} - x^{\mathrm{VI}})^2, \\ &(x' + x'' + x^{\mathrm{V}} - x^{\mathrm{IV}} - x''' - x^{\mathrm{VI}})^2, \\ &(x' + x'' + x^{\mathrm{VI}} - x^{\mathrm{IV}} - x^{\mathrm{V}} - x''')^2, \\ &(x' + x^{\mathrm{IV}} + x''' - x'' - x^{\mathrm{V}} - x^{\mathrm{VI}})^2, \\ &(x' + x^{\mathrm{V}} + x''' - x^{\mathrm{IV}} - x'' - x^{\mathrm{VI}})^2, \\ &(x' + x^{\mathrm{VI}} + x''' - x^{\mathrm{IV}} - x^{\mathrm{V}} - x'')^2, \\ &(x' + x^{\mathrm{IV}} + x^{\mathrm{V}} - x'' - x''' - x^{\mathrm{VI}})^2, \\ &(x' + x^{\mathrm{IV}} + x^{\mathrm{VI}} - x'' - x''' - x^{\mathrm{V}})^2, \\ &(x' + x^{\mathrm{V}} + x^{\mathrm{VI}} - x'' - x''' - x^{\mathrm{IV}})^2. \end{aligned} \]

さて，もしこの方程式の2つの因子が次のように表せると仮定しよう． \begin{align*} &\theta^5-\,f\theta^4\,+\,g\theta^3-\,h\theta^2\,+\,i\theta-\,k\,=0,\\ &\theta^5-f'\theta^4+g'\theta^3-h'\theta^2+i'\theta-k'=0. \end{align*} このとき，\(f\) は上記10個の量のうち5個の和に等しく， \(f'\) は残りの5個の和に等しくなければならない． そして，\(f\) および \(f'\) が 2次の根号のみによって解かれるためには，これらの係数が， \[ y^2 - {M} y + {N} = 0 \] のような2次方程式の根である必要がある． ここで，\({M}\) および \({N}\) は与えられた方程式の係数 \(m, n, p, \dots\) の 有理関数である． したがって， \[ {M} = f + f' \quad \text{および} \quad {N} = f f' \] となる． つまり，\(f\) および \(f'\) の和と積の両方が，係数 \(m, n, p, \dots\) の 有理関数である必要があり，したがって，両者は，根 \(x', x'', x''', \dots\) の どのような置換によっても値が変化しない関数でなければならない． この条件は，和 \(f + f'\) に関しては満たされる． というのも，それは上記10個の量すべての和に等しいからである． しかし，積 \(f f'\) に関してはそうではない． これら10個の量をどのように2つの部分和（各5個）に分けたとしても， それらの積は，根 \(x', x'', x''', \dots\) のあらゆる置換によって 不変であるという性質を持つことは決してないことが容易に確かめられるからである．

あるいは，\(f\) および \(f'\) が同一の2次方程式の根である必要はなく， それぞれが別の方程式に従うという考え方もあるかもしれない． しかし，この例外的な想定を退けるには，以下のように考えれば十分である．\(f\) がある2次方程式 \[ f^2 - {M} f + {N} = 0 \] によって決定されるとする．この方程式の2つの根は，いずれも上記10個の量のうちいずれか5個の和でなければならない[*1]． その2つの和の合計が，根 \(x', x'', x''', \dots\) の置換によって 不変な量 \({M}\) を与える必要がある． この条件が満たされるためには，その2つの和が 上記10個の量すべてを含んでいなければならない． したがって，一方が5個の和であるならば， 他方は必然的に残りの5個の和でなければならないのである．
